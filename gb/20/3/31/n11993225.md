<h6>川普总统3月31日在推特上发布30天停止传播（中共病毒）指南。图为3月28日川普总统在白宫南草坪。(Official White House Photo by Joyce N. Boghosian)
</h6>

	<p>【大纪元2020年04月01日讯】（大纪元记者吴英、徐简、戴芙若综合报导）4月1日全球<ahref="/gb/tag/%E4%B8%AD%E5%85%B1%E7%97%85%E6%AF%92.md#1">中共病毒</a>（<ahref="/gb/tag/%E6%AD%A6%E6%B1%89%E8%82%BA%E7%82%8E.md#1">武汉肺炎</a>）<ahref="/gb/tag/%E7%96%AB%E6%83%85.md#1">疫情</a>最新情况：</p>
<p>中共肺炎全球<ahref="/gb/tag/%E7%96%AB%E6%83%85.md#1">疫情</a>及美国疫情大数据请点击<ahref="/gb/nf1368917.md#1" target="_blank" rel="noopener noreferrer">这里</a>。</p>

<p>美国总统川普（特朗普）3月31日在白宫简报会上表示，美国将面临“非常、非常痛苦的两个星期”。</p>
<p>截至31日，全美<ahref="/gb/tag/%E4%B8%AD%E5%85%B1%E7%97%85%E6%AF%92.md#1">中共病毒</a>确诊数超过18.4万，其中逾7.5万例来自纽约州。全美逾?3,700人死于中共肺炎，逾1,550例在纽约州。</p>
<p>除了美国外，欧洲目前亦是此次疫情的震中，意大利、西班牙、法国、德国等国的确诊数超过5万。</p>
<p>世界银行的最新报告警告，中共病毒可能给东亚和太平洋地区的国家带来经济衰退，并使1,100万人陷入贫困。</p>

<h2>以下是<ahref="/gb/tag/%E6%9C%80%E6%96%B0%E7%96%AB%E6%83%85.md#1">最新疫情</a>实时更新：</h2>

<h3>萨尔瓦多现首例死亡病例</h3>
<p>萨尔瓦多总统纳吉布克（Nayib Bukele）31日在推特上宣布了该国首例冠状病毒患者死亡。在简短的推文中，布基尔说：“上帝会保护我们的。”<br />
<a name="_Toc2020040106"></a></p>
<h3>川普总统发布30天抗疫指南</h3>
<p>川普总统3月31日在推特上发布30天停止传播（中共病毒）指南，内容如下：</p>
<p>遵循州及地方政府的规定；<br />
生病时，请待在家里，不要去工作，并与医疗保健提供者联络；<br />
小孩生病时，不要将他们送到学校，而是让他们待在家里，并与医疗保健提供者联络；<br />
如果家中成员新冠病毒检测呈阳性反应，所有家庭成员都要待在家中，不要出去工作，不要去上学，并与医疗保健提供者联络；<br />
长者要待在家中，远离人群；<br />
有严重慢性病者，感染新冠病毒后风险高，例如肺部、心脏、免疫系统容易受损，因此要待在家中并远离人群。</p>

<h3>纽约市公共运输载量下降九成</h3>
<p>纽约大都会运输局主席兼首席执行官帕特?弗耶（Pat Foye）31日宣布，与疫情爆发之前相比，纽约市大都会运输的载客量下降了90%。</p>
<p>上周末，弗耶的病毒检测结果呈阳性反应，他说，其他582位同事也呈阳性。</p>
<p>弗耶说，有关乘客拥挤事件的报导是零星的，MTA正在与纽约警察局密切合作，控制拥挤情况，并建议乘客改乘其他交通工具或等待下一班车。</p>
<h3>北卡罗莱纳州州长禁止公用事业停止服务</h3>
<p>北卡罗莱纳州州长罗伊?库珀（ Roy Cooper）31日在新闻发布会上说，已签署了一项行政命令，禁止公用事业公司停止对居民提供的天然气、电、水和废水处理等服务。</p>
<p>库珀说，该等服务在未来60天内不能关闭，以使被迫待在家里的居民生活有保障。<br />
<a name="_Toc2020040105"></a></p>
<h3>纽约市确诊病例突破4万 死亡人数破千</h3>
<p>纽约市市长办公室31日晚间发布新闻稿说，截至当天下午6点，纽约市累计确诊病例数为41,771例，死亡1,096人，至少8400人住院，其中至少1888人在ICU。</p>
<h3>内华达州：任何人进入该州都要隔离14天</h3>
<p>内华达州州长史蒂夫?西索拉克（Steve Sisolak）31日发布了旅行警告，要求进入该州的每个人都必须进行14天的自我隔离，即使没有中共病毒的症状。</p>
<p>该建议不适用于从事必要工作的劳工，包括医疗保健和食品供应行业。</p>
<h3>缅因州发布“在家保持健康”令</h3>
<p>缅因州州长珍妮特?米尔斯（Janet Mills）31日发布了一项“在家保持健康”指令，要求该州居民在任何时候都必须待在家里，除非出于必要的工作或其它个人原因。</p>
<p>该命令定于美国东部时间4月2日零时1分生效，并将至少持续到4月30日。</p>
<p>州长还规定了一系列其它限制，包括除非必要，否则禁止使用公共交通工具，并要求继续停课或其它面对面上课的命令，至少直到5月1日。</p>
<p>截至31日，缅因州已确认303例确诊病例，其中5例死亡。<br />
<a name="_Toc2020040105"></a></p>
<h3>荷兰逾千人死</h3>
<p>荷兰死于中共肺炎的人数当日增加了175人，达到1,039人。 荷兰国家卫生研究院（RIVM）表示，确诊数增加845例，至12,595例。<br />
<a name="_Toc2020040105"></a></p>
<h3>新泽西州购买更多冷藏拖车存放尸体</h3>
<p>新泽西州正在命令在全州范围内放置更多的冷藏拖车，以存放尸体。</p>
<p>该州卫生部门通讯官Donna Leusner告诉CNN，他们正在订购冷藏拖车，这些拖车将被放置在该州各地，供州内的医院、疗养院以及体检医师使用，储存尸体。<br />
<a name="_Toc2020040105"></a></p>
<h3>福西：MIT研究不可信</h3>
<p>MIT发表研究报告称，中共病毒飞沫可能传播27英尺远。福西说，这份研究报告偏离事实，不可能会传播这么远，6英尺是合理距离。<br />
<a name="_Toc2020040105"></a></p>
<h3>川普：可以戴口罩或戴围巾 但口罩优先给医院用</h3>
<p>川普总统谈到美国人可以戴口罩或者围巾以阻止中共病毒的传播。至于是否建议所有美国人都戴口罩时，他说，所生产的口罩可以送到需要的医院。</p>
<p>“我们希望做好的口罩是被送去医院，福西博士今天告诉我，我们不希望每个人都在与医院竞争医疗用品。”</p>
<p>“因此，可以根据需要使用围巾，或者其它东西，不一定要戴口罩，但是至少在一段时间内，这不是一个坏主意。”</p>
<p>“我的意思是说，最终不会需要这样做，也不必这样做。它会消失的，它将会消失，希望会消失很长时间。”川普说。<br />
<a name="_Toc2020040104"></a></p>
<h3>川普：未来两周将非常痛苦</h3>
<p>川普总统31日表示，全美要为即将到来的中共病毒病例激增做好准备，未来两个星期将“非常非常痛苦”。</p>
<p>他还说，这不是流感，是“恶毒”的病毒，其传染性非常强，可能很多医生都不清楚这点，这是非常可怕的情况。</p>
<p>川普说，如果没有在早期采取中国旅行限制令，美国可能会面临更多的死亡人数，可能会看到有人在酒店大厅倒下，或者在飞机上倒下。<br />
<a name="_Toc2020040103"></a></p>
<h3>白宫顾问：全美死亡人数预估10万到24万</h3>
<p>白宫特别工作组协调员黛博拉?伯克斯（Deborah Birx）博士和安东尼?福西（Anthony Fauci）博士31日说，他们仍预计全美将有10万到24万人死于中共肺炎，接下来两周的死亡人数将达到高峰。</p>
<p>福西博士说，中共肺炎的致死率是流感的10倍，美国应该做好准备，因为可能有10万名美国人死于中共病毒。</p>
<p>“尽管这个数字令人震惊，但我们应该为此做好准备。会那么多吗？我希望不会，而且我认为只要我们越努力，就越能缓解，这个死亡人数就越有可能减少。”他说。</p>
<p>他强调，这是预测的数字，我们不接受它，我们将尽一切可能做到这一点。<br />
<a name="_Toc2020040102"></a></p>
<h3>英国一名13岁男孩死于中共肺炎</h3>
<p>英国一名13岁男孩成为该国最年轻的受害者。国王学院医院说，一名13岁男孩检测结果呈阳性反应，且已死亡。据信他是英国这次疫情中最年轻的受害者。<br />
<a name="_Toc2020040105"></a></p>
<h3>CDC：四种情况患者染疫风险高</h3>
<p>美国疾病控制和预防中心（CDC）说，患有糖尿病、慢性肺病、心脏病或吸烟的人如果感染中共病毒，患严重并发症的风险可能会增加。</p>
<p>美国疾病预防控制中心（CDC）在其第一份报告中研究了可能使COVID-19恶化的潜在健康状况，分析了在2月12日至3月28日期间，来自所有50个州和美国四个地区的确诊病例数据。CDC检查了7,162例基本的健康状况或其他潜在的危险因素。<br />
<a name="_Toc2020040101"></a></p>
<h3>法意西英单日死亡人数创新高</h3>
<p>31日，法国、意大利、西班牙、英国的单日死亡人数创新高。</p>
<p>英国的死亡人数从过去24小时的1,408例增加了381例，达到1789例，同比增长27%，是迄今为止最大涨幅。意大利的死亡人数则增加了837，达到12,428例。西班牙增加到8464例。</p>
