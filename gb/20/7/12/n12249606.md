<tr><td VALIGN=TOP width="626"><h2 align=center>想从维州进入南澳？一文看懂新规定</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2020/06/sa-border-600x400.jpg" />
<h6>由于近期维州的中共病毒疫情恶化，南澳持续关闭与维州的边境。(AAP)
</h6>
<hr>
<p>【大纪元2020年07月12日讯】（大纪元记者戴宁澳洲阿德莱德编译报导）由于近期墨尔本的<ahref="/gb/tag/%E4%B8%AD%E5%85%B1%E7%97%85%E6%AF%92.md#1">中共病毒</a><ahref="/gb/tag/%E7%96%AB%E6%83%85.md#1">疫情</a>不断恶化，<ahref="/gb/tag/%E5%8D%97%E6%BE%B3.md#1">南澳</a>持续关闭与<ahref="/gb/tag/%E7%BB%B4%E5%B7%9E.md#1">维州</a>的边境。如果您需要从维州进入<ahref="/gb/tag/%E5%8D%97%E6%BE%B3.md#1">南澳</a>，下面的信息可以帮助您了解相关的新限制规定。</p>
<ul>
<li>
<h2>为什么在南澳和<ahref="/gb/tag/%E7%BB%B4%E5%B7%9E.md#1">维州</a>之间要实行这些<ahref="/gb/tag/%E8%BE%B9%E5%A2%83%E9%99%90%E5%88%B6.md#1">边境限制</a>？</h2>
</li>
</ul>
<p>这些<ahref="/gb/tag/%E8%BE%B9%E5%A2%83%E9%99%90%E5%88%B6.md#1">边境限制</a>是由于维州的<ahref="/gb/tag/%E4%B8%AD%E5%85%B1%E7%97%85%E6%AF%92.md#1">中共病毒</a>病例激增所致。通过加强南澳与维州之间的边境限制，可使<ahref="/gb/tag/%E7%96%AB%E6%83%85.md#1">疫情</a>扩散至南澳的风险降到最低。</p>
<ul>
<li>
<h2>从维州到南澳的道路会封闭吗？</h2>
</li>
</ul>
<p>从维州进入南澳的道路，无论是铺好的道路还是未铺设的道路，都会设有一个静态的检查站或硬路障，以阻止人们进入。警方将在南澳的边境地区增加巡逻，并与澳洲国防军协商以获得协助。</p>
<ul>
<li>
<h2>在南澳和维州之间的哪条道路上设有硬路障？</h2>
</li>
</ul>
<p>南澳警方仍在确定哪些道路上须设有硬路障。</p>
<ul>
<li>
<h2>南澳与维州之间边境的新限制将实施多长时间？</h2>
</li>
</ul>
<p>这些硬性边界限制措施将实行多久仍未可知，定期会对其进行监查<strong>。</strong></p>
<ul>
<li>
<h2>我在维州，希望进入南澳。可以吗？</h2>
</li>
</ul>
<p>只有在维州的南澳居民才允许进入南澳。返回南澳者须自我隔离14天。</p>
<p>在维州的非南澳居民须具有“必要旅行者”的身份，才可进入南澳。但是，当他们不执行“必要旅行者”职责时，必须隔离并穿戴个人防护设备。</p>
<ul>
<li>
<h2>对州际公路的卡车司机有哪些限制？</h2>
</li>
</ul>
<p>南澳卡车司机如须在边境上往来（无论是否去维州），返回南澳时不必自我隔离。他们要尽量减少在其它州所逗留的时间，并且未经事先批准不得进入残疾人护理、教养、保健或老年护理机构。</p>
<p>维州卡车司机（包括任何从事该运输或货运服务的工作人员），要以商业方式提供进出南澳州内、出入南澳州的运输或货运服务，在公共场合应始终佩戴口罩，并在不履行职责时进行隔离。</p>
<p>新州和首都地区的卡车司机进入南澳无须进行自我隔离，但他们必须尽量减少在南澳逗留的时间，并且未经事先批准不得进入残疾人护理、教养、保健或老年护理机构。</p>
<ul>
<li>
<h2>我是卡车司机，我可以携带一名家庭成员作为乘客乘坐我的卡车吗？他们进入南澳必须进行自我检疫吗？</h2>
</li>
</ul>
<p>如果某人满足“必要旅行者”身份，他们将获得“必要旅行者”的旅客表格。乘客（例如孩子、伴侣、朋友或家人）不被视为“必要旅行者”，须在南澳指定地址进行14天的自我隔离。</p>
<ul>
<li>
<h2>我可以在哪里进行自我隔离？</h2>
</li>
</ul>
<p>您必须直接前往您选择的自我隔离地点，并在那里待14天。</p>
<p>您必须确保没有其他人进入或逗留在您进行隔离的地方，除非：</p>
<p>*他们通常住在此地，而您需要给他们提供护理或支持，或从他们那里获得支持或护理；或者</p>
<p>*他们也进行自我隔离；或者</p>
<p>*您的住所已进行适当设置，因此您不会与其他人有所接触。</p>
<ul>
<li>
<h2>我已经从维州进入南澳，是否必须接受病毒检测？</h2>
</li>
</ul>
<p>这不是强制性的，但是来自维州的人们被要求在到达后的24小时内进行病毒测试，以及在其14天自我隔离期的第12天再次进行病毒检测。但是，如果您有以下症状，无论多么轻微，都要进行病毒检测：发烧或发冷、咳嗽、喉咙痛、流鼻涕、呼吸急促、味觉或嗅觉丧失。</p>
<ul>
<li>
<h2>我是“必要旅行者”，已经提交了跨境旅行登记，但尚未获得我的身份确认，我仍能进入南澳吗？</h2>
</li>
</ul>
<p>如果您已经完成了跨境旅行登记，请启动前往南澳的旅行计划，并携带所有相关文件和证据以备向南澳警方出示。到达南澳后，南澳警方将对“必要旅行者”核实身份证明和旅行目的。</p>
<ul>
<li>
<h2>我是南澳居民，想从维州回家可以吗？</h2>
</li>
</ul>
<p>可以。南澳人可以返回家中，但需要自检14天。</p>
<p>您必须能够证明南澳是您的居住地。</p>
<ul>
<li>
<h2>我农场的一半在南澳，另一半在维州，我们可以使用禁止通行的道路进入我们的农场吗？</h2>
</li>
</ul>
<p>居住在南澳和维州边境的初级生产者，在前往其物业的一部分以检查存货或从事农业活动时，可以使用这些在其它情况下禁止通行的道路。这些道路不向社区其他成员开放。</p>
<ul>
<li>
<h2>我是维州居民，想来南澳可以吗？</h2>
</li>
</ul>
<p>不能。维州居民除非是“必要旅行者”，否则不得进入南澳。</p>
<ul>
<li>
<h2>如果有人违反规定，会怎样处理？</h2>
</li>
</ul>
<p>任何人如被发现违反相关规定，都将受到有关当局的警告和/或罚单。</p>
