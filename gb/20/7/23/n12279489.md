<tr><td VALIGN=TOP width="626"><h2 align=center>疫情下人们消费习惯有所改变</h2>

<h6></h6>
<hr>
<p>【大纪元2020年07月24日讯】( 大纪元记者章丰<ahref="/gb/tag/%E6%96%B0%E8%A5%BF%E5%85%B0.md#1">新西兰</a>编译报导 ) 一项调查显示，近 60 % 新西兰人在<ahref="/gb/tag/%E4%B8%AD%E5%85%B1%E7%97%85%E6%AF%92.md#1">中共病毒</a> ( Covid -19 ) 下的<ahref="/gb/tag/%E5%B0%81%E9%94%81.md#1">封锁</a>状态后受到<ahref="/gb/tag/%E7%BB%8F%E6%B5%8E.md#1">经济</a>影响。</p>
<p>仅 4 月份就有近 4 万人失业，随着失业率飙升，又有数千人需要政府的财政帮助。与 3 月份相比，4 月份的工作岗位减少了 37,500 个，这是自 20 年前统计局开始统计工作岗位以来的最大降幅。</p>
<p>凯维萨沃的安保公司 ( AMP ) 调查了人们在<ahref="/gb/tag/%E5%B0%81%E9%94%81.md#1">封锁</a>前和封锁后对自己财务状况的看法。</p>
<p>AMP 财富管理公司首席执行官布莱尔 · 弗农 ( Blair Vernon ) 说 : “ 近 60 % 受访者表示，他们已经遭受了<ahref="/gb/tag/%E7%BB%8F%E6%B5%8E.md#1">经济</a>影响，工作时间减少或收入减少等情况。”</p>
<p>来自<ahref="/gb/tag/%E6%96%B0%E8%A5%BF%E5%85%B0.md#1">新西兰</a>零售业的格雷格 · 哈福德 ( Greg Harford ) 说，随着流入家庭的资金减少，以及就业的不确定性，消费支出放缓是很自然的。</p>
<p>在过去的八个星期，与去年同期相比，消费者在零售和非零售商品上的每周支出都有所增加。</p>
<p>4 级警报期间人们的支出大幅下降，但在 5 月初，平均每笔交易的支出为 54 纽元，而 2019 年同期平均每笔交易的支出为 42 纽元左右。</p>
<p>有 70 % 的人表示，他们现在会做出财务上的改变，如减少支出。而近 60 % 的人会在其他支出的基础上保持原有支出。</p>
<p>只有四分之一受访者知道自己退休后的养老金余额是多少，40 多岁的人是被激励采取行动的大群体之一，比如向理财顾问咨询。</p>
<p>利率将在一段时间内保持在非常低的水平，但对于那些想存钱的人来说，无论存钱的原因是什么，都建议咨询后再做打算。</p>
<p>责任编辑：蓝克</p>
