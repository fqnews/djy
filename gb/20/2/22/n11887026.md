<p>【大纪元2020年02月22日讯】大纪元每天为读者梳理<a href="/gb/tag/%E7%BF%BB%E5%A2%99%E5%BF%85%E7%9C%8B.md">翻墙必看</a>的文章：</p>
<p>1 . <a href="/gb/20/2/21/n11886484.md">【新闻看点】一线医生揭疫情内幕 习称拐点未到</a><br />
澳洲学者冯崇义表示，<a href="/gb/tag/%E5%8C%97%E4%BA%AC.md">北京</a>当局在<a href="/gb/tag/%E6%AD%A6%E6%B1%89%E8%82%BA%E7%82%8E.md">武汉肺炎</a>爆发后，祭出了诸如封锁信息、隐瞒疫情、欺骗民众和没有配套措施的封城等昏招，造成举世罕见的灾祸。当局错失病患初发时有效控制疫情的黄金时间，剥夺民众的知情权，一门心思营造盛世幻象，使疫情迅速蔓延，祸及世界。</p>
<p>2. <a href="/gb/20/2/21/n11886654.md">周晓辉：病毒入北京核心区 中南海还不警醒</a><br />
无疑，病毒在中共权力核心区域肆意妄为，中南海更需要警醒，因为这是对其所为最严厉的警告，即警告中南海妄想以人力“战胜”疫情、“战胜”病毒是行不通的，而是要反省所犯下的罪恶，并勇于清算，才可获得上天的垂怜，否则若还不警醒，最终只能走向无尽的深渊。</p>
<p>3. <a href="/gb/20/2/21/n11884783.md">【独家】遭中共报复 前大陆医生存款被冻结</a><br />
去年12月19日，曾就职于浙江大学附属医院的钟医生（化名），向《大纪元时报》讲述了自己在实习期间参与浙大附一院肝移植手术的亲身经历（【独家】《大陆医生亲历不寻常器官移植手术（上）》），曝光中共涉活摘良心犯器官牟利的罪行。然而，就在两周后的2020年1月7日，钟医生突然发现，他在国内名下所有银行存款都遭到非法冻结，微信及支付宝功能受限，钟医生的父亲也受到了牵连。</p>
<p>4. <a href="/gb/20/2/21/n11884463.md">北京多家医院爆发新冠肺炎群聚感染</a><br />
在新冠状病毒肺炎（俗称<a href="/gb/tag/%E6%AD%A6%E6%B1%89%E8%82%BA%E7%82%8E.md">武汉肺炎</a>）疫情持续升温下，首都<a href="/gb/tag/%E5%8C%97%E4%BA%AC.md">北京</a>多家医院出现疫情，引关注。有网传消息称，北京的防疫升到武汉级别，但消息无法得到证实。</p>
<p>5. <a href="/gb/19/11/29/n11689892.md">政府公务员因修炼法轮功遭迫害致死综述（4）</a><br />
据明慧网资料不完全统计，自1999年7月中共开始迫害法轮功至今，大陆政府机关公务员系统中的法轮功学员，至少有137人遭政法委、“610”迫害致死或在迫害中离世。</p>
<p>6. <a href="/gb/20/2/21/n11884649.md">中共多家监狱爆发大规模新冠肺炎疫情</a><br />
新冠状病毒肺炎（俗称武汉肺炎）疫情持续蔓延，中共多家监狱爆出大规模疫情。中共官方自称，山东任城监狱、浙江省十里丰监狱及湖北监狱系统共有数百人确诊。目前山东、浙江、湖北数名官员被免职。但外界普遍质疑中共官方通报的数据，认为实际感染人数比官方公布的数字多出很多。</p>
<p>7. <a href="/gb/20/2/21/n11884399.md">【一线采访】为腾床位 武汉肺炎病患被强制出院</a><br />
“治愈数据水分很大！”武汉张女士的母亲日前被医院强制出院后，病情加重，而分区看病让她们再难求一床。大量的感染病人、并发症患者都在苦等社区安置，有的在绝望中自杀。武汉封城被指造成了无可挽回的人道灾难。</p>
<p>8. <a href="/gb/20/2/21/n11884557.md">山东维权人士逃美：曾被中共做药物试验</a><br />
山东维权人士界立建，虽然今年仅有32岁，但已历经多次被中共抓捕、2次被扔进精神病院受药物摧残的血泪史。近日接受采访时，他曝光了被关在精神病院期间被迫成为药物测试的“白老鼠”内幕，以及亲眼所见访民“被消失”的悲剧。</p>
<p>9. <a href="/gb/20/2/21/n11884970.md">为何武汉病毒实验室无法平息阴谋论？</a><br />
武汉病毒研究所的P4病毒学实验室一直是阴谋论的风暴眼，美国《石英》（Quartz）杂志周四（2月20日）指，这也是新冠状病毒爆发以来公众对中共政府信任程度急剧下降的又一表现。</p>
<p>10. <a href="/gb/20/2/21/n11886708.md">非常时期 江西上千民众聚集市政府讨血汗钱</a><br />
2月20日、21日，上千名“罗氏百姓金行”的受害者聚集在江西省宜春市的市政府前讨血汗钱。因该金行法人代表罗军涉嫌非法吸收公众存款，2月19日晚间投案自首，金行被警方查封，涉案金额达4亿元（人民币，下同）。</p>
<p>11. <a href="/gb/20/2/21/n11885487.md">美国作家12年前“预言”武汉肺炎爆发</a><br />
美国作家布朗（Sylvia Browne）在12年前出版的一本书中写道，类似肺炎的严重疾病将于2020年在全球大流行，这似乎预言了现今武汉肺炎的爆发。</p>
<p>12. <a href="/gb/20/2/21/n11886401.md">外媒：不要让中共主导UN机构 世卫是警讯</a><br />
《日经亚洲评论》2月20日刊登评论文章说，不要让中共开始主导联合国机构，西方国家和日本必须阻止北京按照它的造型重塑世界。</p>
<p>13. <a href="/gb/20/2/21/n11886677.md">【唐靖远快评】北京2月底须控制疫情的秘密</a><br />
换言之，修改后的诊断标准，客观上成为复工的一大松绑令，给政府的强制复工铺平了道路，只要患者不确诊，企业就必须撸起袖子加油干。临床诊断标准的诊断权在医生手中，政府不好控制，现在核酸检测诊断权在政府手中，那就很好办了，一切都是党说了算。</p>
<p>14. <a href="/gb/20/2/21/n11886124.md">北京驱逐华日记者 专家揭中共的误导手法</a><br />
中共驱逐《华尔街日报》三名驻华记者，引发全球关注。《华盛顿邮报》发表评论文章认为，尽管驱逐的表面原因是北京对《华日》的一篇文章不满，但还有一个更有可能的解释。</p>
<p>15. <a href="/gb/20/2/21/n11886721.md">政治局称疫情得到初步遏制 中共专家唱反调</a><br />
中共政治局宣称，新冠肺炎疫情得到“初步遏制”，但中共疾控中心流行病学首席科学家曾光却唱反调，认为湖北省外确诊病例可能会“再上升”。</p>
