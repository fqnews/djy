<p>【大纪元2020年01月25日讯】大纪元每天为读者梳理<a href="/gb/tag/%E7%BF%BB%E5%A2%99%E5%BF%85%E7%9C%8B.md">翻墙必看</a>的文章：</p>
<p>1 . <a href="/gb/20/1/24/n11819273.md">湖北党报记者吁武汉“一把手”下台 遭处理</a><br />
对于<a href="/gb/tag/%E6%AD%A6%E6%B1%89%E8%82%BA%E7%82%8E.md">武汉肺炎</a>疫情肆虐，大陆舆论指责中共当局应对迟缓。中共湖北省委机关报《湖北日报》高级记者张欧亚24日发表言论说，武汉市政府应立即换帅，当日其单位致函武汉市委市政府道歉，称张欧亚的言论给各级领导“添了堵”。</p>
<p>2. <a href="/gb/20/1/24/n11818934.md">【新闻看点】武汉肺炎失控 医护人员绝望痛哭</a><br />
面对接连不断的病患，一线医护人员非常辛苦，也最危险。有医护人员在微信群里哭诉，实际情况可怕的多，估计“有十万病人”，告诫亲人全靠自救。视频中显示，武汉医护人员在放声大哭。有的医生在微信中说，“我们已经顶不住了”，病人跪求救救他们，而所有的防护用具都用光了……</p>
<p>3. <a href="/gb/20/1/22/n11813874.md">见证大法神奇 大陆民众恭贺李大师新年好</a><br />
在2020年中国传统新年来临之际，大陆民众纷纷向海外明慧网寄来贺词、贺卡，衷心表达对李洪志大师节日的祝福和感恩。</p>
<p>4. <a href="/gb/20/1/24/n11818893.md">内幕：中共疾控中心操控试剂盒 控制疫情数据</a><br />
武汉疫情大爆发导致全面封城。据知情者披露，中共国家卫健委公布的数据，由检测新型冠状病毒的试剂的结果决定，但中共的<a href="/gb/tag/%E7%96%BE%E6%8E%A7%E4%B8%AD%E5%BF%83.md">疾控中心</a>通过试剂盒发放量来“控制”感染人数与死亡率，跟实际疫情没有关系，手法非常邪恶。</p>
<p>5. <a href="/gb/20/1/24/n11818530.md">武汉医院告急 向外求援 医护哭诉：顶不住了</a><br />
武汉封城进入第二天，多家医院医用物资告急，纷纷向外求援。大量医护传出的信息显示，不少医院已经弹尽粮绝。有些医护人员发帖指一天都不能吃东西。死了很多人也没有人收尸，有的医护放声痛哭。</p>
<p>6. <a href="/gb/20/1/18/n11801851.md">大纪元2020庚子年新年贺词</a><br />
回首旧岁，逢九之年，风云动荡，中共式微，中国势奇，正合中共应劫、中华转机之意。看国内，经济颓废、百业萧条，爆雷滚滚、危机四伏，中共破船独飘零；观国际，贸战日酣、西方渐醒，香港奋起，各国齐援，全球反共掀高潮。</p>
<p>7. <a href="/gb/20/1/24/n11819156.md">何坚：生化危机？武汉封城的动机推演</a><br />
一座逾千万人的大都市，“新萨斯”发源地——武汉，封城了。中共最新的封城举措，不但震惊了世界，更将千万名武汉居民推入致命的恐慌中。电影<a href="/gb/tag/%E7%94%9F%E5%8C%96%E5%8D%B1%E6%9C%BA.md">生化危机</a>正在现实中上演，中共封城之举，透射出惊悚杀机。</p>
<p>8. <a href="/gb/20/1/23/n11817075.md">中国人群起揭美中贸协真相 中共文宣转态</a><br />
美中贸易战打了近两年，终于在1月15日签署了第一阶段贸易协议，暂时停火。虽然中共官方和媒体大力吹捧此次签署结果“平等”、“互利”，但舆论大多认为，此次北京做出的很大的让步。而大陆网民也纷纷揭露中共让步的真相。</p>
<p>9.<a href="/gb/20/1/10/n11783110.md">【内幕】一文看懂 中共为何急推数字货币</a><br />
步入2020，中国面对贸易战和经济下滑的阴云，从海外投资者到国内的企业家及中产们，都在为自己的财富寻觅出路。与此同时，伴随着美国针对中共从盗窃技术到侵犯人权等各种恶行频频制裁，越来越多中共相关企业担忧被美国制裁波及。中共央行今年异常激进地推动数字货币，透露出其欲借科技之力进行最后一搏。</p>
<p>10. <a href="/gb/20/1/23/n11816088.md">网络删文 钟南山闭嘴 疫情下中共急控言论</a><br />
武汉新型冠状病毒疫情完全失控，中共当局也开始大力控制舆论走向。日前首先披露<a href="/gb/tag/%E6%AD%A6%E6%B1%89%E8%82%BA%E7%82%8E.md">武汉肺炎</a>“会人传人”中共专家钟南山，传他已闭嘴，有陆媒媒体人在国内网站发文被删除。有香港专家指，此次武汉肺炎感染规模，至少是当年SARS的十倍。</p>
<p>11. <a href="/gb/20/1/24/n11818924.md">疫情失控 全大陆旅游暂停 故宫迪士尼关闭</a><br />
武汉肺炎疫情失控，全大陆仅剩西藏未传疫情。中共官方24日公告暂缓旅游活动；北京故宫博物院、上海迪士尼等也宣布自25日起关闭。</p>
<p>12. <a href="/gb/20/1/24/n11819131.md">王友群：牢记历史教训 不要替中共挡枪子</a><br />
如今，中共已大祸临头。“天灭中共”不只是一个口号，正在变成每个人都看得见的现实。作为医治“武汉肺炎”的医护人员，最重要的是认清中共的邪恶本质，不上中共的当，不要盲目听从中共的冷血号召，不要做无谓的牺牲。在尽量多救人的同时，一定要保护好自己。</p>
<p>13. <a href="/gb/20/1/24/n11819134.md">专家：新冠状病毒亲属间传播率可高达83%</a><br />
武汉肺炎正迅猛扩散，香港知名传染病专家袁国勇等多名专家，在医学杂志《柳叶刀》（香港译作：刺针）发表文章，指新型冠状病毒在亲属之间传播率非常高。</p>
<p>14. <a href="/gb/20/1/24/n11818570.md">武汉现第四代病例 医生对家人说数万人感染</a><br />
北京时间1月24日凌晨2点，世界卫生组织（WHO）发布的最新声明中披露，中国已经向其报告了“武汉的第四代病例和武汉以外的第二代病例”，但未进行进一步阐述。多个武汉医护人员告诉家人的视频爆料：医生估计大概有九万、十万人感染，千万不要相信政府，得靠自己。</p>
<p>15. <a href="/gb/20/1/24/n11817804.md">大陆各界民众祝福大法 感念李洪志先生恩德</a><br />
2020庚子年中国新年到来，法轮大法（法轮功）传世即将迈入第28年，大法的传出对当今中国及世界产生深远影响。值此佳节之际，中国大陆的各界民众通过大纪元向法轮大法创始人李洪志先生拜年，并献上他们诚挚的祝福与问候。</p>
