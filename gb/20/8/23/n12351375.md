<tr><td VALIGN=TOP width="626"><h2 align=center>【Realmark珀斯房地产专栏】拍卖售房优于私下销售吗？</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2020/08/us-HongKong-officials-sanction-ccp_1200x800-320x200.jpg" />
<h6></h6>
<hr>
	<p>【大纪元2018年10月30日讯】您是否想知道拍卖和私人签约出售房产之间的不同？ 本系列中，我们将从买家和卖家两个角度与您分享这些信息。</p>
<p>西澳房地产经纪公司Realmark董事John Percudani 谈他喜欢以拍卖会售房的理由：“拍卖能把买家召集在一起，拍卖会总是热烈并富有激情。”他说。尽管多数西澳人相信拍卖能成功，但找到对拍卖有信心的房地产经纪人却不那么容易。</p>
<h2 style="font-weight: inherit;"><strong><span style="font-style: inherit;">拍卖胜于私下销售</span></strong></h2>
<p>作为卖方，您得依赖于经纪人为您定的市场价格，其中市场走向及市场情绪起主导作用。因此，有时需等很久，看不到尽头。而买方对其他买家的出价却不知情，这可能导致以低于您预期的价格出售。</p>
<p>拍卖会则不同，您不必透漏要价，但可在上市28天内完成销售。拍卖会上所有买家都来到现场，您最终可以定夺是否脱手。</p>
<h2 style="font-weight: inherit;"><strong><span style="font-style: inherit;">经纪人是关键</span></strong></h2>
<p>拍卖能否成功取决于经纪人。他们必须专业，并有扎实的运作系统和程序，需在短期内做大量工作。经纪人协调拍卖时需检查：</p>
<ul>
<li style="font-style: inherit; font-weight: inherit;">◎透明度。需随时回答卖方、拍卖商和买方的问题；</li>
<li style="font-style: inherit; font-weight: inherit;">◎对截止期要严格遵守并有有效程序；</li>
<li style="font-style: inherit; font-weight: inherit;">◎数据完整，以便卖方了解情况；</li>
<li style="font-style: inherit; font-weight: inherit;">◎及时向拍卖师报告详情。</li>
</ul>
<h2 style="font-weight: inherit;"><strong><span style="font-style: inherit;">这意味着什么？</span></strong></h2>
<p>作为卖方，您聘请的专业人员应该知道怎样拍卖。您会接到目前市场信息详情，确切了解您家房子的位置和价格。在拍卖会上，您应跟进每一步了解竞拍结果是什么。</p>
<p>作为买方，在拍卖会当天，您需和专业人员谈一谈该物业，以全面掌握情况。</p>
<p>拍卖会幕后的专业人员了解这一地区的特征，也知道卖方情况，以及哪些人是买家。买家会和拍卖师、卖家认真交谈。</p>
<h2 style="font-weight: inherit;"><strong><span style="font-style: inherit;">拍卖小窍门</span></strong></h2>
<h3>对卖家：</h3>
<ul>
<li style="font-style: inherit; font-weight: inherit;">◎观察倾听竞拍情况；</li>
<li style="font-style: inherit; font-weight: inherit;">◎查一查会上有几个买家；</li>
<li style="font-style: inherit; font-weight: inherit;">◎看一看竞拍幅度；</li>
<li style="font-style: inherit; font-weight: inherit;">◎看一看叫价间隔。</li>
</ul>
<p>从以上各方面可发现您物业的市场热度、买家心态及有多少钱“已抓到手了”。现场竞拍至终结，价格呈透明态。</p>
<h3>对买家：</h3>
<ul>
<li style="font-style: inherit; font-weight: inherit;">◎不要做神秘买家，把您自己介绍给代理人和拍卖会；</li>
<li style="font-style: inherit; font-weight: inherit;">◎坐在拍卖师和其他竞拍者看得见的位置上；</li>
<li style="font-style: inherit; font-weight: inherit;">◎早期出价，免得措手不及；</li>
<li style="font-style: inherit; font-weight: inherit;">◎不要等待别人先出价，尽快参与进来玩一把；</li>
<li style="font-style: inherit; font-weight: inherit;">◎报价清晰、果断。</li>
</ul>
<p>“以强有力的肢体语言和清晰的声音告诉他人：您要赢，您不会退缩。那您可能就会赢。 ”Percudani说。</p>
<p>本专栏由珀斯房地产公司Realmark提供，更多房产拍卖详情请联系公司房地产销售持牌人及董事总经理John Percudani，电话：(08) 9328 0900，电邮：<ahref="mailto:jpercudani@realmark.com.au">jpercudani@realmark.com.au</a></p>
