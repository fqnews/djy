  <p>五行理论，简略示下：《参考内经第五篇 》</p>
  <p>肝：属木 = 属风 = 属酸 = 属东 = 苍色 = 肝生筋 = 在体为筋 = 在脏为肝 = 其窍为目 = 在志为怒 = 怒伤肝 = 与心连</p>
  <p>心：属火 = 属热 = 属苦 = 属南 = 赤色 = 心生血 = 在体为脉 = 在脏为心 = 其窍为舌 = 在志为喜 = 喜伤心 = 与脾连</p>
  <p>脾：属土 = 属湿 = 属甘 = 属中 = 黄色 = 脾生肉 = 在体为肉 = 在脏为脾 = 其窍为口 = 在志为思 = 思伤脾</p>
  <p>肺： 属金 = 属燥 = 属辛 = 属西 = 白色 =肺生皮毛= 在体为皮毛= 在脏为肺 = 其窍为鼻 = 在志为忧 = 忧伤肺</p>
  <p>肾： 属水 = 属寒 = 属咸 = 属北 = 黑色 =肾生骨髓= 在体为骨 = 在脏为肾 = 其窍为耳 = 在志为恐 =恐伤肾</p>
  <p>上述之五行理论可观人体之金、木、水、火、土之理，东、西、南、北、中之方位，风、热 、湿、燥、寒的性，酸、苦、甘、辛、咸之味，苍、赤、白、黑、黄之色。基于这些基本概念，衍生出更细腻判症方向与定夺治症药物的配伍。</p>
  <p><ahref="/gb/tag/%E8%8D%AF%E8%86%B3.md#1">药膳</a>亦然。不过较好看也美味得多了，比煎熬成浓黑的药汁，要美妙很多。</p>
  <p><b>中国常用说法有医学依据</b></p>
  <p>人常说某人心肝很黑，或是恶向胆边生，胆边生毛，或是肝胆向照，或是怒火中烧，乐极生悲等等用词。其实是有医学根据的。因为当人体在恶念起时，胆囊就启起化学运动。胆汁增浓加多，使胆囊膜绷得较紧。放大镜下观之，胆汁充斥胆囊膜组织。看上去，很像胆边生了毛…</p>
  <p>胆又与肝脏唇齿相依。所以当胆囊变色，肝即随之起舞，同时呈现苍黑色，正如肝胆相照。在怒火中烧时，则是肝先变色，胆亦随之起毛，绝不背叛！而此时，心脏老兄得急速充血来侍候肝与胆，急急之下即速转呈为红黑色。所以心肝很黑。</p>
  <p>具上述词句型之人，多数得胆炎、胆结石或肝疾病，当然被传染则不在此列。同时兼有心脏疾病的危机。尤其突来的大喜, 心脏未先接收念的指令, 于是就发生了… 总之，常起恶念、怒火常烧的朋友，一定要先固肝保胆。爱方城之战的朋友，竹战之前，务必携带… 救心！(以上言论　纯粹摘述)</p>
  <p>《黄帝内经》第二篇尾段提到：“圣人不治已病治未病，不治已乱治未乱，夫病已成而后药之，乱已成而后治乱，譬犹渴而穿井，斗而铸锥，不亦晚乎？”人真的不能等到病了，才寻找医治，在未病时，即应于生活起居间，调理自身健康，预防疾病发生。民以食为天。</p>
  <p><b><ahref="/gb/tag/%E8%8D%AF%E8%86%B3.md#1">药膳</a>的分类</b></p>
  <p>药膳，不愧是人类续天命、<ahref="/gb/tag/%E5%85%BB%E7%94%9F.md#1">养生</a>、补益之善方妙法。纵是如此，在制与食药膳之前, 要知目的为何而食？食何种药膳？具体可分为四大类：</p>
  <p><一> 因病症而制而食。这一类又分七大因：</p>
  <p>( 1 ) 袪风：疏散外风＝用菊花、甘草；平熄乾风 = 天麻钩藤白蜜饮；凉血清热 ＝ 松子桑椹粥；凉血熄风＝莴苣炒藕片</p>
  <p>( 2 ) 清热：清热解毒＝凉拌浦公英；清热泻火＝丝瓜汤；清热凉血＝软炸牡蛎；清热；滋阴＝人参枸杞酒</p>
  <p>( 3 ) 袪湿：清热利湿＝凉拌三苋；健脾渗湿＝三色糯米饭</p>
  <p>( 4 ) 化痰：健脾燥湿＝黄耆茯苓粥　</p>
  <p>( 5 ) 化瘀：山楂酒、牛肝粥、生化汤等</p>
  <p>( 6 ) 补益：益气＝淮药芝麻糊；养血＝红枣当归饮；滋阴＝首乌鸡蛋；补阳＝麻辣羊肉炒葱头</p>
  <p>( 7 ) 补气：行气＝人参酒<br />须由执业中医师判症之后，而配伍之膳食，属治疗性质。 </p>
