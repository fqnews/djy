<tr><td VALIGN=TOP width="626"><h2 align=center>10种男性不可忽视的症状</h2>

<h6></h6>
<hr>
<p>【大纪元2011年08月10日讯】（大纪元记者陈珍妮编译报导）最近《今日美国》（USA Today）刊登一项新调查发现，约50%男性身体不适不去看医生。或许是男性自古以来被赋予强者形象，总是将身体的不适不当回事，认为忍一忍就能过去，所以不需要看医生。然而男性的不适，有25%会尽可能的拖延就医，17%至少会拖延一个星期。但许多疾病的造成都是因对小病的忽视而变成大病的。</p>
<p>专家提醒，并不是任何的不适症状都须就医，但有些看似无关紧要的信号忽视不得。美国健康医疗网《健康之友》（Woman&#8217;s Day）列出10种男性不得忽视的严重症状，并呼吁女性须更加关切家中男性的健康问题，并协助他们就医。</p>
<p><B> ( 1 ) 说话时舌头打结</B><br />忽视：被当成是健忘症。<br />提醒：可能是中风前兆。当血栓或血凝块堵塞血管、血管壁损伤及血管狭窄时，会阻碍脑部的血液循环，致使脑部供血不足而出现“短暂性脑缺血发作”（TIA )，因而造成脑部功能受损及说话困难，俗称“小中风”。出现这种症状时须挂急诊紧急治疗。小中风通常仅持续几分钟，症状在一小时内即消失，但这是更严重的中风的凶险前兆。男性发生TIA比率较女性稍为高些，而且会随着年龄增高，尤其55岁以后概率更高。</p>
<p><B> ( 2 ) 呼吸短促</B><br />忽视：认为是身体状况不佳。<br />提醒：可能是心脏病发作的征兆。例如爬楼梯时呼吸急促、费力，可能是心脏肌肉（心肌）得不到足够的氧气供应的象征。这是冠心病（冠状动脉硬化型心脏病）的特征，男性55岁起患病率增高。</p>
<p>美国加州大学戴维斯分校，泌尿科的男士健康计划联合负责人，辛德尔医师（Alan Shindel）提醒，任何原因引起的呼吸困难（包括气喘与过敏症），均会因费力活动而恶化，如果病情一次次加重，很可能是心脏问题。应该就医照一个心电图、超音波或胸部X光。如果呼吸吃力情况无改善有必要打求救电话。</p>
<p><B> ( 3 ) 腹部一侧持续疼痛</B><br />忽视：认为是肌肉拉伤。<br />提醒：可能是肾结石或肿瘤。卡普蓝（Kaplan）医师提醒，如果是持续性的隐约作痛，则须就医，由医师来判断，腹痛的原因有许多，但医师首先会考量是否是肿瘤的可能，任何腹部的疼痛持续3天未见改善，应该就医检查。如果突如其来、令人无法忍受的剧痛一小时仍未改善，赶紧挂急诊。他表示，如果是肾结石的疼痛，会从一个特定的点辐射到它处，而且剧烈疼痛伴随呕吐。也可能是盲肠炎、溃疡或疝气。</p>
<p><B> ( 4 ) 如厕出血</B><br />忽视：认为是食物引起红色尿液。<br />提醒：如果是尿液带血，可能是肾结石或膀胱癌；如果是粪便带血，则可能是结肠癌。若便后擦拭时卫生纸上有鲜红血迹，可能是痔疮。但若血便变细如铅笔般，有可能是癌症征兆。粪便黑如柏油状（又称黑便），可能是胃溃疡出血的象征。</p>
<p>有些食物（如甜菜根或红肉火龙果）会使尿液及粪便呈现短暂红色。但如厕出现任何的红色均轻忽不得，有可能是血。美国罗德岛米瑞安医院（Miriam Hospital）男士健康中心联合负责人麦诺医师（Martin Miner）提醒，甚至是极小的血量也是警讯。</p>
<p><B> ( 5 ) 勃起困难</B><br />忽视：认为没关系有药可吃。<br />提醒：可能是心血管疾病的预警。麦诺医师表示，最新研究显示，男性在40至50岁之间出现勃起功能障碍（ED）者，可能 2~5年内得心血管疾病。近期梅约诊所（Mayo Clinic）的一项研究发现，男性在40多岁患ED者，心脏出现问题风险比无ED者高出50倍。可能是动脉硬化所引起，最好找泌尿科医生做心电图或都卜勒超音波测试生殖器血流。</p>
<p><B> ( 6 ) 白天疲惫</B><br />忽视：认为晚上早点睡即可。 <br />提醒：可能是睡眠时呼吸道狭窄或阻塞，使呼吸与睡眠中断，造成睡眠断断续续，无法得到充分休息，可能每小时会干扰睡眠5到30次之多，致使白天疲累、嗜睡，甚至开车容易睡着。即因“阻塞性睡眠呼吸中止症”，又称阻塞性睡眠呼吸暂停症或阻塞性睡眠窒息症。辛德尔医师提醒，此症发作会造成心脏病、心律不整、中风或心脏衰竭的机会，常见于过胖和严重打鼾。可以采取减肥、手术扩大呼吸道和在夜间使用呼吸治疗设备。</p>
<p><B> ( 7 ) 持续情绪暴躁</B><br />忽视：认为是工作压力大的关系。<br />提醒：可能是忧郁症。遇到工作上的困难或琐碎事时，会烦躁不安与情绪低落是正常现象，但通常几天便消失，而临床忧郁症（clinical depression）的悲观、烦躁症状每次会持续几个星期。辛德尔医师表示，劝他去看心理医生，并让他知道这几个星期以来，他的行为变得不像你所认识的他，或他对过去原有兴趣的活动已失去兴致，或者工作表现不佳等。</p>
<p><B> ( 8 ) 频尿</B><br />忽视：认为是喝太多咖啡引起。<br />提醒：可能是第2型糖尿病或摄护腺肥大。如果是咖啡、茶等利尿饮料引起的，只须少喝即能避免夜间多尿。但若时常一个晚上需要起床解尿超过二次，可能是身体试图将因无法进入细胞内，而在血液中堆积过多的葡萄糖（即血糖）由尿液排出，即所谓的“糖尿病”，故造成多尿、夜间频尿并经常口渴。糖尿病通常可以控制，例如注意饮食、运动和药物治疗。</p>
<p>再者，摄护腺肥大亦会造成频尿。摄护腺（前列腺）位于膀胱下方，围绕着负责将尿液自膀胱内排出的尿道。摄护腺肥大会压迫尿道，导致排尿机能障碍，如频尿、夜尿、滴尿、排尿迟缓、尿不干净等，是中老年人的普遍现象。男性50岁以下的频尿，可能起因于泌尿系统感染。亦有可能是摄护腺癌，尤其是年过50男性发生概率增加。</p>
<p><B> ( 9 ) 皮肤发黄</B><br />忽视：不清楚什么是皮肤发黄。<br />提醒：可能是肝脏问题。肝功能不好会出现皮肤发黄或黄疸。成人黄疸可能是肝脏疾病、胆结石、胰线癌或病毒性肝炎感染，造成肝脏肿大以致引发黄疸。肝脏最常见的疾病为肝炎，出现症状有黄疸、发烧、胃部不适、食欲差及疲累等。</p>
<p><B> ( 10 ) 皮肤长新痣、斑点、红斑点</B><br />忽视：过些时候会消失。<br />提醒：可能是皮肤癌或脂溢性角化病（又称老年疣、基底细胞乳头状瘤），为中年人常见的一种蜡状、聚集产生的良性皮肤肿块。如果皮肤长出不明肿块，或者原有肿块形状、颜色改变或变大时，应该看医生。尤其当斑点或痣出现颜色变深、流血、发痒、疼痛时，美国皮肤医学会提醒这可能是皮肤癌的征兆。良性肿块与皮肤癌外观上难以区别，尤其是恶性黑色素瘤。</p>
<p><B>为什么男人不肯看医生？</B><br />男性看医生比例比女性低24%。一项对1,000名男性的调查显示，有强烈传统男子气概观念的男性（比如认为男子汉不能抱怨病痛），接受预防保健比率较无此观念的男性少一半。男性不看医生的另一个原因是，不肯承认自己生病，因为他“害怕”。</p>
<p><B>如何引导、协助另一半去看医生？</B><br />晓之以理，动之以情。坐下来好好的与他谈，让他了解定期健康检查的重要性，能及早发现体内潜藏的健康问题及早治疗，以防止情况恶化。并强调他的健康对你与这个家庭的重要性。研究显示，这种方法是让男性愿意看医生的最强动力，或许是因为吸引了他们传统男性气慨角色的思想吧。</p>
<p>如果必要，替丈夫约见医生，以表示你对他的健康的关切，同时能起到激励、推动作用，亦能维护他的尊严（因为男性认为求助是弱者表现）。就医前，先替他写下问题与担忧的事。若他希望你陪伴同行，帮他记录医生提醒的重要讯息。因为人于焦虑不安时，往往听不清对方说的话。</p>
