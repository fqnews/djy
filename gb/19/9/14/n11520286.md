<p>【大纪元2019年09月14日讯】大纪元每天为读者梳理<a href="/gb/tag/%E7%BF%BB%E5%A2%99%E5%BF%85%E7%9C%8B.md">翻墙必看</a>的文章：</p>
<p>1 .<a href="/gb/19/9/13/n11519711.md"> 李嘉诚反驳政法委指责：遗憾 习惯莫须有指责</a><br />
香港首富<a href="/gb/tag/%E6%9D%8E%E5%98%89%E8%AF%9A.md">李嘉诚</a>日前呼吁政府要对“未来主人翁”网开一面，被<a href="/gb/tag/%E4%B8%AD%E5%85%B1%E6%94%BF%E6%B3%95%E5%A7%94.md">中共政法委</a>点名指责“纵容犯罪”。李嘉诚基金会发声明回应，称李嘉诚对其言论被曲解感到遗憾。</p>
<p>2. <a href="/gb/19/9/13/n11518812.md">中秋月圆 逾40国法轮功学员问候李洪志师父</a><br />
在中秋月圆、举家团圆之时，海外法轮功学员思念、感恩他们的师父李洪志先生，敬祝他中秋快乐。自1992年起法轮功洪传于世27年，恩泽神州大地，福祉五湖四海，“真善忍”的光辉照耀寰宇。<br />

<p>4. <a href="/gb/19/9/13/n11519729.md">【新闻看点】中共吹“世纪联姻”英断然拒绝</a><br />
香港证券交易所（HKEX）前天主动向伦敦交易所“天价提亲”，计划用316亿英镑并购。权衡之后，伦敦交易所今天（9月13日）断然回绝了这门“亲事”，指出这桩“世纪联姻”看不到任何好处。</p>
<p>5. <a href="/gb/19/9/13/n11519201.md">中共政法委批李嘉诚 部署抢夺富豪资产</a><br />
“希望年轻人能体谅大局，执政者能为未来的主人翁网开一面。”<a href="/gb/tag/%E6%9D%8E%E5%98%89%E8%AF%9A.md">李嘉诚</a>日前出席慈山寺活动时，表达了对当前香港局势的建言。一句“网开一面”招来中共的刀把子和笔杆子双管齐下对李开火，同时，中共释出港版“公私合营”信息。</p>
<p>6. <a href="/gb/19/9/13/n11519951.md">被政治化的月饼 两女富豪UN发言代表谁？</a><br />
香港人过中秋节，大多数人买大班或美心月饼，但今年因为反送中运动和中共的抹黑及打压，月饼被赋予了政治化的意味。</p>
<p>7. <a href="/gb/19/9/13/n11519686.md">中共间谍威胁在德抗议者 各界吁将其驱逐出境</a><br />
中共当局正通过间谍对在德国的异见人士、示威者进行监控与恐吓，其中包括支持香港民主运动的活动参与者。德国各界呼吁联邦政府采取措施，让这些“中共间谍”承担后果，甚至可以将其驱逐出境。</p>
<p>8. <a href="/gb/19/9/13/n11519627.md">组图：9.13和你拖手太平山灯光人链</a><br />

<p>9. <a href="/gb/19/9/13/n11519362.md">【新闻看点】深感危机？贸易战北京突然放软</a><br />
如果不出意外，美中第13轮贸易谈判下月初将在华盛顿举行。还有20天左右的时间，中方接连向美方示好，甚至试图采用“双轨”谈判。外界认为，北京突然对贸易战放软身段，可能是已经感受到了危机。</p>
<p>10. <a href="/gb/19/9/13/n11519340.md">中秋节 大陆异见人士家属寄语狱中亲人</a><br />
在中秋佳节阖家团圆之际，被中共迫害入狱的众多异见人士与亲人，仍承受着离别的痛苦。中秋夜，部分家属通过大纪元表达对狱中亲人的思念和问候，并向给予关心帮助的朋友们送去节日的祝福。</p>
<p>11. <a href="/gb/19/9/13/n11517988.md">中共下令国企加强投资香港 控制多个行业</a><br />
香港反送中抗争已经持续了3个多月，中共加紧对香港布局。路透社13日引述消息称，中共要求国有企业加大在香港投资，以及加强于不同范畴企业的控制。</p>
<p>12. <a href="/gb/19/9/13/n11519258.md">习亲信接掌 张晓明疑被免港澳小组副组长</a><br />
香港反送中运动高涨，敏感之际，中共中央港澳工作协调小组人事变动，习近平亲信、公安部长赵克志出任港澳小组副组长，港澳办主任张晓明疑似被免去副组长职务。</p>
<p>13. <a href="/gb/19/9/13/n11520112.md">香港人启动B计划 转移资产或移民他国</a><br />
香港的抗议活动从夏季延伸到秋季，似乎仍没有解决的迹象。香港人申请移民他国的数量激增，表明更多当地人正计划离开香港。</p>
<p>14. <a href="/gb/19/9/13/n11518782.md">蓝营学者蔡金树遭中共羁押已超过一年</a><br />
台湾实践大学教授江岷钦爆料，南台湾两岸关系协会联合会主席蔡金树，因“国安”理由，已遭中共关押，据了解蔡金树自从去年7月已遭羁押，迄今已1年近2个月约419天，只知目前人被关在福州某看守所。</p>
<p>15. <a href="/gb/19/9/11/n11514533.md">【花意诗心】桂花 冠绝中秋月里来</a><br />
总觉得，如果没有且清且甜而又不绝如缕的独特清香，如果这清香不是从“一点宫黄”中“情疏迹远”而又柔柔地飘出来，便没有桂花的神与魂。但这好像并不是有意做出来的行为，而是一种境界，一种自然而清静淡远的境界。</p>
