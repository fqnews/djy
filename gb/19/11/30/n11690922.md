<tr><td VALIGN=TOP width="626"><h2 align=center>【翻墙必看】中共担心的蝴蝶效应来了</h2>

<h6></h6>
<hr>
<p>【<a href="/gb/tag/%E5%A4%A7%E7%BA%AA%E5%85%83.md">大纪元</a>2019年11月30日讯】大纪元每天为读者梳理<a href="/gb/tag/%E7%BF%BB%E5%A2%99%E5%BF%85%E7%9C%8B.md">翻墙必看</a>的文章：</p>
<p>1 . <a href="/gb/19/11/29/n11690272.md">【新闻看点】北京最担心的来了 西方随美制裁</a><br />
对港警的暴恶，西方国家终于看不下去了。继美国总统川普向香港人送出感恩节大礼――签署《香港人权与民主法案》生效为法律后，其它西方国家也有了行动。正在或准备效仿美国，对侵犯人权的中港两地官员制裁举动，中共担心的蝴蝶效应正在显现。</p>
<p>2. <a href="/gb/19/11/29/n11689585.md">大纪元准确传达香港民意 呈现时代真相</a><br />
在烽火前沿，<a href="/gb/tag/%E5%A4%A7%E7%BA%AA%E5%85%83.md">大纪元</a>和香港抗议民众站在一起，与红媒形成鲜明对照。记者们深入社区，与青年学子、社运人士、公务员、银发族、前任警花和大陆移民等各界市民互动倾谈；大纪元客观公正的报导赢得了港人的信任和赞赏，据记者介绍，在抗争现场，很多民众握着他们的手含泪感谢，感谢大纪元在危难中陪着他们向前走。</p>
<p>3. <a href="/gb/19/11/29/n11690413.md">【新闻看点】香港法中共强烈反弹 一部门沉默</a><br />
美国总统川普签署《香港人权与民主法案》。对此，中共恼羞成怒，令不少人士怀疑，可能会影响到美中贸易谈判，认为美中协议可能会告吹。但是在中共“七连发”的扰攘中，经常谈论美中关系的商务部，一直没有发声，直到现在仍然保持着沉默。</p>
<p>4. <a href="/gb/19/11/28/n11686633.md">王立强事件分析 是谁害怕大众知道真相？</a><br />
中共特工王立强上周末向澳洲投诚，其指证的向心夫妇随后亦在台湾被扣查，中共对此事也罕见做出快速回应。而网络上各种复杂说法纷呈。原首都师范大学副教授李元华接受了大纪元采访，谈了他对这些现象的分享，并厘清了事实。</p>
<p>5. <a href="/gb/19/11/27/n11683728.md">从时间节点看中共如何抹黑香港抗争者（上）</a><br />
从6月至今，香港人一直在抗争。通过这场运动，香港人也越来越认清中共，“天灭中共”的标语遍布香港大街小巷。针对这场运动，中共媒体不断掩盖此期间发生的事实、抹黑抗争者，比如选择性曝光、责任推给他国、情绪化评论等等。</p>
<p>6. <a href="/gb/19/11/28/n11687804.md">感恩节 纽约腰鼓队法轮功学员谢师尊（上）</a><br />
“师尊您好：我们是纽约法轮大法腰鼓队的全体大法弟子，在感恩节来临之际，向您敬上最谦恭、虔诚的问候：‘师尊好！恭祝师尊节日快乐！您为救度我们，辛苦了！’”纽约法轮大法腰鼓队的全体大法弟子在感恩节之际，向法轮功创始人李洪志先生表达敬意和感恩。</p>
<p>7. <a href="/gb/19/11/29/n11690483.md">掌掴男子 央视女记者孔琳琳在英国获罪</a><br />
中共央视驻伦敦记者孔琳琳去年在英国被指控普通袭击罪，周五（11月29日），一名英国法官判处她罪名成立。</p>
<p>8. <a href="/gb/19/11/29/n11690630.md">打麻将牵出200亿家产 省高院女副院长被免</a><br />
因一起敲诈案导致其200亿家产暴露的海南省高院副院长张家慧，落马5个月后被免职。这起敲诈案却是同张家慧爱打麻将有关。</p>
<p>9. <a href="/gb/19/11/29/n11690079.md">斯里兰卡新政府索要租给中共99年的海港</a><br />
斯里兰卡把港口租给中共99年抵债一事，引起全球对中共借“一带一路”项目扩张的担忧。近日该国新政府以国家利益为由，希望撤销前政权将汉班托塔南部港口出租给中企的决定。</p>
<p>10. <a href="/gb/19/11/29/n11690517.md">中俄在玩火？ 将与伊朗首次三国联合军演</a><br />
伊朗宣布很快将与中国（中共）和俄罗斯举行首次海上联合军事演习。这一举动被解读为三国联合向西方，尤其是美国展示团结力量和秀肌肉。但有分析警告说，伊朗牌并不好打，中俄与伊朗过于走近甚至可能引火烧身。中共和俄罗斯迄今对联合军演的反应非常低调。</p>
<p>11. <a href="/gb/19/11/29/n11690519.md">中国经济真相 分析：实际GDP近乎官方一半</a><br />
中共官方经济数据虚假已是业界共识，人们更关注的是中国经济真相。近日有专业人士深入分析说，中共政府驱动经济的模式不再奏效，现在中国经济面临债务高企、投资扭曲等风险，中国经济警报已经拉响，且遍及各个领域。</p>
<p>12. <a href="/gb/19/11/29/n11689958.md">向心龚青疑用本名购屋 20万月租给网路新贵</a><br />
中共间谍“王立强”揭露中国创新投资公司董事向心及其妻子龚青是中共高级情报官员，主要负责渗透台湾的任务等。接受检调单位调查时，向心坦承他在台北市信义区豪宅“冠德远见”购有2屋；地籍誊本曝光后，发现夫妻2人疑似是用3个字的本名登记，而非借名购买。</p>
<p>13. <a href="/gb/19/11/29/n11688657.md">中共介入选举？韩国瑜避答新唐人提问引质疑</a><br />
中共间谍“王立强”出逃案，揭露中共介入台湾选举的细节，国民党总统参选人韩国瑜27日在宜兰接受媒体联访，新唐人记者问道，是否认为中共介入选举？是否愿意呼吁中共不要介入选举？韩国瑜数度跳针，反问哪家媒体，却没有回答记者提问，这段画面在台湾多个脸书粉丝团以及政论节目上被讨论。</p>
<p>14. <a href="/gb/19/11/29/n11690576.md">夏小强：为何川普总统成为“香港英雄”？</a><br />
11月28日，，大批香港民众参加在中环爱丁堡广场举行的感恩节集会，感谢川普签署《香港人权与民主法案》。集会在晚上约10时结束，大会宣布有约10万人参加。</p>
<p>15. <a href="/gb/19/11/29/n11690451.md">脸书中国员工比例高 引发中共渗透的忧虑</a><br />
脸书（Facebook）创办人扎克伯格（Zuckerberg）近日批评中共的网路言论审查制度，还表示放弃打入中国市场。但是脸书目前雇用许多中国员工，他们是脸书最大的群体，一些人还支持北京立场，引外界担忧，这些人来美国却被中共洗脑？疑替中共做事。</p>
