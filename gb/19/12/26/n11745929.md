<p>2.<a href="/gb/19/12/25/n11745406.md"> 中共发文严防规模性失业抗争 预示失业潮？</a><br />
12月14日，<a href="/gb/tag/%E4%B8%AD%E5%85%B1.md">中共</a>国务院发文件，称要“全力防范化解规模性失业风险”，“各地区要第一时间处置因规模性失业引发的群体性突发事件”的消息引发各界关注。</p>

<p>5. <a href="/gb/19/12/25/n11745408.md">陆企逾1300亿违约创纪录 官称明年更难</a><br />
大陆今年以来债务违约规模创下新高，近70家企业的170多只债券发生违约，逾1300亿的违约金额超过去年全年规模，有多家企业年内多次违约，涉及国企和民企。中共央行官员承认，明年债券市场“形势严峻”，违约风险更高。</p>
<p>6. <a href="/gb/19/12/25/n11744649.md">英预言家：2020年中国发生革命 川普连任</a><br />
曾经成功预言川普（特朗普）当选美国总统的英国预言家汉密尔顿-帕克（Craig Hamilton-Parker）表示，在即将来到的2020年，川普将会连任总统职务，而中国则会发生革命。</p>
<p>7. <a href="/gb/19/12/24/n11742729.md">曾遭40种酷刑 医生伍群再被劫入永川监狱</a><br />

<p>9. <a href="/gb/19/12/25/n11745711.md">揭抚顺丑闻后 花千芳又剑指广州遭讥讽</a><br />

<p>10. <a href="/gb/19/12/25/n11745733.md">大陆44家民企变国企 国资入股业绩受关注</a><br />

<p>11. <a href="/gb/19/12/25/n11745144.md">廖祖笙：杀人党血腥“执政”70年</a><br />

<p>12. <a href="/gb/19/12/25/n11745357.md">李小龙女儿起诉大陆真功夫餐厅 索赔逾2亿元</a><br />

<p>13. <a href="/gb/19/12/25/n11745538.md">杨宁：炮制圣诞节起源谎言 中共用心险恶</a><br />
毋庸置疑，中共禁过圣诞节，就是要否定上帝的存在，否定圣诞精神，阻止人们追随上帝的脚步，懂得爱，懂得宽恕和分享。这与禁止其他信仰的目的是一致的，其最终目的就是让没有其他信仰的中国人，接受中共的歪理邪说。</p>
<p>14. <a href="/gb/19/12/25/n11745304.md">谈中共频闪避 韩国瑜获“倒赞”数创新高</a><br />
台湾总统大选进入倒数阶段，各党总统候选人近几个月接连与热门网红合作，成功引发话题。国民党总统候选人韩国瑜日前接受脱口秀节目《博恩夜夜秀》专访，主持人单刀直入地问：“现在有哪个国家最危害到台湾安全？”，韩国瑜一时语塞，随后结巴地说，希望两岸能和平交流，并未正面回应。</p>
<p>15. <a href="/gb/19/12/25/n11745594.md">分析：价值观冲突令中国品牌国际地位低</a><br />
《金融时报》专栏作家刘裘蒂12月25日发表文章表示，中国和西方国家的价值观冲突，包括是否尊重言论自由，以及是否尊重隐私和窃取机密等方面，令中国品牌的国际地位进一步下降。</p>
