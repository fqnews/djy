<tr><td VALIGN=TOP width="626"><h2 align=center>郭微言：迎神韵  观神韵</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2020/03/GettyImages-1188708561-320x200.jpg" />
<h6></h6>
<hr>
	<p>【大纪元2月23日讯】<b>引人瞩目到处未演先轰动</b></p>
<p>美国<ahref="/gb/tag/%E7%A5%9E%E9%9F%B5.md#1">神韵</a>艺术团首次在2004年新年晚会的纽约百老汇上演，一举成名，随后在美国五个城市巡回演出 ; 2006年获美国专业的杂志BILLBOARD票选为北美第七名的大秀。2007年，从年初开始作巡回演出，在短短的四个个多月时间内，足迹遍布世界各地三十多个城市，最后在5月16日加拿大爱民顿市圆满结束公演，总计将近八十场，二十多万现场观众，创下国际艺术界的的大震撼。</p>
<p>今年全球巡回快速扩增到近70个城市，共220场，预估65万人次观赏。<ahref="/gb/tag/%E7%A5%9E%E9%9F%B5.md#1">神韵</a>圣诞晚会从2007年12月18日开锣，首先在纽约百老汇碧肯剧院(Beacon Theatre) 演出10场，另外，巴尔地摩理瑞克歌剧院也同时开演，分别由神韵纽约艺术团和神韵巡回艺术团担纲。</p>
<p>神韵风靡纽约百老汇 ; 大西洋城 ; 费城 ; 波士顿 ; 洛杉矶 ; 亚特兰大 ; 拉斯维加斯 ; 底特律 ; 佛罗里达圣彼得堡、劳德岱堡 ;巴尔的摩 ; 夏洛特 ; 格林斯伯勒 ; 达拉斯..等等，最后在纽约曼哈顿无线电城音乐厅压轴再爆满15场，震撼7万观众，终于在2月10日圆满结束在美国的演出。紧接着，继续在包括德国、法国、荷兰、比利时等等在内的十几个欧洲国家巡回演出。另外一团，从日本开演，转往韩国、台湾、马来西亚、澳大利亚、新西兰等亚洲各地巡回演出。到处未演先轰动，晚会所到之处，佳评如潮。</p>
<p>神韵晚会旨在弘扬人类的正统文化、神传文化，弘扬博大精深的中华文化、道德文化，激浊扬清，净化心灵。</p>
<p><b>高评价的热烈回响</b></p>
<p>温妮莎?夏伍德（Vanessa Harwood）曾是加拿大国家芭蕾舞团的首席舞蹈演员，于1984年赢得了加拿大国民的最高荣誉奖——加拿大国家勋章。夏伍德表示，她将向国际舞蹈联盟（World Dance Alliance）的成员推荐神韵晚会。她评价表示：“晚会很优美，不只是漂亮的那种美，而是很严谨端庄的那种美 — 美的很有深度，美的很有内涵 ; 专业水准有很高，做得非常好。”</p>
<p>名列国际顶级大提琴师前30名的瓦列芙斯卡(Christine Walevska)，在12月18日、21日两度观看过新唐人圣诞晚会。她称赞演出是“晚会的音乐、舞蹈、天幕、整个的演出都是顶级、一流的。这样子的演出是百看不厌的。”</p>
<p>泰德?柯凡诺(Ted Kavanau)是CNN的创办人之一，也是资深媒体制作人，他获得多次艾美奖及无数新闻专业的其他奖项。他表示：“我最喜欢蒙古舞，其实，每一个节目我都很喜欢，这是最伟大的，最富有专业水准的演出 ; 整台晚会棒极了！难以想像的好 ; 是第一流水准的。”</p>
<p>加拿大歌剧团的首席号手琼恩?威思滕（Joan Weston），1月18日，她和家人及艺术界的友人一起观看在多伦多的神韵演出。她赞叹说：“我对歌唱非常敏感，今天的歌唱绝对是世界一流的。当然舞蹈也是亮丽鲜艳，毫不逊色。”</p>
<p>在纽约华人家长会21周年年会获得“ 教育领袖奖”的美东中文学校纽约理事黄金利博士观看神韵后表示，“用这样的高水准的艺术展现出仁义礼智信的文化道德内涵。中华文化赋予华人有包容，谦虚的美德并关爱他人。新唐人晚会的节目就体现出真诚，善良和包容的理念，并体恤他人苦难，维护公义，希望这样好的节目应尽量广泛让人们知道。”</p>
<p>新泽西青年交响乐团的创始人和指挥Barbara Barstow夫人表示：“这一场演出是如此的美妙(fabulous)，是真正的壮观(Spectacular)！我特别喜欢戚晓春演奏的二胡。节目涵盖很广，很有包容性，而且做得如此雅致( so tastefully done！)，非常难得 ; 两位主持人表现得非常好（Very cute！Well done！) ; 很幽默 ; 很有娱乐性(Very entertaining！) 。”</p>
<p><b>口碑载道百闻不如一见</b></p>
<p>中国舞，特别注重身韵、身法，讲究手、眼、身、法、步，更要求形、神、劲、律的展现，体现的是中华正统文化、神传文化的根本－“信仰为本，道德为尊”。神韵演出阵容是如此的强大，而且歌唱、演奏、编曲、舞蹈方方面面都体现出是出自于非常高造诣的艺术家之手。演员们的手、臂、头、脚等的动作非常丰富而多彩多姿，一举手、一投足，不可思议的细腻、不可思议的美 ; 每个细节都很不同寻常，传神、传情、韵味非常独特，令人目不暇接。</p>
<p>神韵演员有一种极深刻的内在美，演员的气质和涵养的境界都溶入在艺术里，精、气、神各个角度都达到正、善与美相当高的境界，是神形兼备又饶富韵味的境界。天幕、音乐、舞蹈、和每个场景搭配的恰如其分，如同梦幻一般，都是一流的水准的。舞台天幕场景的美轮美奂、辉煌壮观，宛如身历其境就在再现的大唐盛世之中。观众看了都频频赞叹 : 撇开高水准的诠释内涵不谈，光是看舞姿、服装、天幕三者之一，就非常值得了。</p>
<p>男女歌唱家的歌声十足扣人心弦，余音绕梁，是震撼的呼唤，也是心灵的净化。透过精湛艺术的表现而诠释神的传说、意像、寓意以及正统文化的内涵、风韵、音律，让人观赏了这至善至美、寓意深远、典雅高贵的中国舞，有如沐春风的愉悦、舒畅和溶入的共鸣。</p>
<p>神韵艺术团所体现的艺术之美，所传递高境界的讯息，不仅赢得舞蹈音乐 绘画的艺术界专业人士的激赏与赞叹，更难得而重要的是，同时，她还搏得全世界各地不同民族、不同文化、不同信仰、不同工作领域、背景 ; 不分老少的观众，人传人，口传口，喜闻乐见的一睹为快，赏心悦目的快乐，福至心灵，乐在其中。＠<br /><font color=#ffffff>(http://www.dajiyuan.com)</font><br /><center><font class=GY13>本文只代表作者的观点和陈述</font></center></p>
