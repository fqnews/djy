<tr><td VALIGN=TOP width="626"><h2 align=center>选民关心头等大事 看安省各党派政纲如何响应</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2018/06/IMG_0459-600x400.jpg" />
<h6>6月7日，安省将诞生42届新省政府。（滕冬育/大纪元）
</h6>
<hr>
<p>【大纪元2018年06月01日讯】（大纪元记者李平编译报导）<ahref="/gb/tag/%E5%AE%89%E7%9C%81%E7%9C%81%E9%80%89.md#1">安省省选</a>日益逼近，选民最关心的头等大事都有哪些呢？CBC新闻此前读者调查显示，选民最关心的5项头等大事包括：民生，医疗，教育,电价和交通。围绕这些问题，自由党、保守党、新民主党和绿党推出的竞选政纲都有哪些呢？一起来看看。（注意：随着竞选日益推进，各党可能会更多政纲推出，一起来随时关注）。</p>
<h2><strong><b>民生</b></strong></h2>
<p><strong><b>自由党</b></strong></p>
<ul>
<li>2020年实现2岁半~4岁儿童免费全天托儿服务</li>
<li>提高17%高收入人群（相当于近180万人）个税率，5万及以上年入多交税168元，13万及以上年入多交税近200</li>
<li>近68万人个税略降，多数人个税基本不变</li>
<li>5%企业税维持不变</li>
<li>明年最低时薪从14元涨至15元</li>
<li>消除车保费率地区差异</li>
<li>提高烟草税至每条烟征税4元</li>
</ul>
<p><strong><b>保守党</b></strong></p>
<ul>
<li>提供托儿退税，低入家庭托儿退税最多抵销4分之3托儿开支，15万及以上年入家庭退税最多抵销4分之1托儿开支</li>
<li>将中等收入个税省税从现有的15%削减至7.32%</li>
<li>暂停最低时薪上调，维持现有14元最低时薪不变。省税减税优惠，相当于低入群体个税为零。</li>
<li>企业税从现有的5%减至10.5%</li>
<li>中小企业税减3%</li>
<li>汽油税中的省税每升减10分，废除安省碳税计划</li>
<li>废除禁止酒厂1元啤酒售价的立法</li>
</ul>
<p><strong><b>新民主党</b></strong></p>
<ul>
<li>年入4万以下家庭享受托儿全免，其它多数家庭托儿费每天12元左右</li>
<li>明年最低时薪从14元涨至15元</li>
<li>企业税从现有的5%涨至13%</li>
<li>22万及以上年入人群个税涨1%，30万及以上年入人群个税涨2%</li>
<li>9万以上豪华车收3%附加税</li>
<li>制定年度炒房投机税，打击不作任何税务贡献的外国和国内炒房投机者</li>
<li>车保减15%</li>
<li>赋予安省能源署更大权力，如油价调控权，禁止节假日长周末油价乱涨</li>
</ul>
<p><strong><b>绿党</b></strong></p>
<ul>
<li>实施全民基本收入削除贫困</li>
<li>明年最低时薪从14元涨至15元</li>
<li>1%顶尖收入群体个税涨1%</li>
<li>下调中小企业工资税</li>
<li>企业税从现有的5%涨至12.5%</li>
<li>实施物业空置税</li>
</ul>
<p><strong><b>医疗</b></strong></p>
<p><strong><b>自由党</b></strong></p>
<ul>
<li>今年医疗和基建拨款22亿，聘 3500名护士，到2022年省内每家长期护理院配1名注册护士</li>
<li>精神健康医疗拨款21亿</li>
<li>去年秋曾承诺新增1200个医院床位，今年年内兑现</li>
<li>未来5年内新增5000个长期护理床位，未来10年内新增3万个长期护理床位</li>
<li>为无单位牙科保险的成人和老人提供牙科保险，最多报80%的费用</li>
<li>老人纳入OHIP+医疗保险</li>
</ul>
<p><strong><b>保守党</b></strong></p>
<ul>
<li>每年投入9800万补助低入老人牙科护理</li>
<li>未来4年新增5万个长期护理床位</li>
<li>精神健康医疗拨款19亿</li>
<li>北方和农村地区医生享受省税减税优惠</li>
</ul>
<p><strong><b>新民主党</b></strong></p>
<ul>
<li>未来10年医疗和基建拨款190亿</li>
<li>医院拨款每年追加3%</li>
<li>立即新增2000个医院床位</li>
<li>未来5年新增5万长期护理床位，未来10年新增4万个长期护理床位</li>
<li>无论就业与否，所有人，包括无牙科保险的低入老人和儿童，均享受牙科全保，</li>
<li>75亿全民药保计划，125种处方药全保</li>
<li>上台后第1年聘4500名护士</li>
</ul>
<p><strong><b>绿党</b></strong></p>
<ul>
<li>未来4年精神健康医疗拨款41亿，最后逐渐将之纳入OHIP+医保</li>
<li>无牙保工人享受药保和牙保</li>
<li>现有药保计划延伸至老人</li>
<li>追加农村地区Local Health Integration Networks拨款</li>
</ul>
<h2><strong><b>教育</b></strong></h2>
<p><strong><b>自由党</b></strong></p>
<ul>
<li>教育基金拨款25亿，特殊教育拨款3亿</li>
<li>按计划继续推动和扩大低中收入家庭子女免大学学费</li>
<li>保留和更新现有性教育课程</li>
</ul>
<p><strong><b>保守党</b></strong></p>
<ul>
<li>废除和修订性教育课程</li>
<li>取消现有教学大纲中的发现式数学课程，恢复传统数学课程</li>
<li>允许安省高等教育质量委员会（HQCO）调查高等院校言论自由被侵犯事件</li>
</ul>
<p><strong><b>新民主党</b></strong></p>
<ul>
<li>学校基建拨款160亿</li>
<li>保留和更新现有性教育课程</li>
<li>重新评估教育拨款公式</li>
<li>幼儿园每个班人数限定在26人</li>
<li>取消标准化考试</li>
<li>大学学贷拖欠累计利息全部勾销</li>
<li>符合经济补助的大学生以学习补助取代学贷</li>
<li>职业技术学徒计划拨款5700万</li>
</ul>
<p><strong><b>绿党</b></strong></p>
<ul>
<li>为需经济补助的大学生提供无息贷款</li>
<li>重新评估特殊教育拨款优先项</li>
<li>取消标准化考试</li>
<li>4~8年级班级学生人数限定在22人</li>
<li>追加低收入社区高中拨款</li>
</ul>
<h2><strong><b>电费</b></strong></h2>
<p><strong><b>自由党</b></strong></p>
<ul>
<li>保留现有公平电力计划中的25%返还计划，农村地区最多可享受40~50%电费返还</li>
</ul>
<p><strong><b>保守党</b></strong></p>
<ul>
<li>省第一电力公司红利返还纳税人，将现有公平电力计划返还再增12%</li>
<li>解散第一电力公司董事会，设立被解雇董事遣散费上限</li>
</ul>
<p><strong><b>新民主党</b></strong></p>
<ul>
<li>回购第一电力公司出售给私人的股份，收回第一电力公司政府控股权，取消时段电费计划，电费最多降30%</li>
<li>向渥京争取取消电费HST税</li>
</ul>
<p><strong><b>绿党</b></strong></p>
<ul>
<li>废除公平电力计划，购买魁省更便宜的水电</li>
<li>实施碳税与红利计划</li>
<li>关闭和停运核电厂</li>
</ul>
<h2><strong><b>交通</b></strong></h2>
<p><strong><b>自由党</b></strong></p>
<ul>
<li>对照联邦拨款追加多伦多交通项目（包括市中心地铁舒缓线）拨款，拨款上限90亿</li>
<li>拨款110亿修多伦多至温莎高铁</li>
<li>Presto 交通卡用户GO火车和UP Express所有票价一律3元，多伦多以外乘客10公里以内票价3元</li>
<li>将省交通项目建设燃油税中市镇划拨比例从现有每升2分追加至2021年的每升4分</li>
</ul>
<p><strong><b>保守党</b></strong></p>
<ul>
<li>多伦多地铁项目拨款50亿，省府控制 TTC基建</li>
<li>完成渥太华轻轨下期项目，资助其它地区地区交通计划</li>
<li>恢复Northlander火车服务</li>
<li>保留省府燃油税中市镇现有划拨比例</li>
</ul>
<p><strong><b>新民主党</b></strong></p>
<ul>
<li>拨款修建多伦多市中心地铁舒缓线，全省交通和辅助客运净成本补贴50%，拨款修建渥太华轻轨下期项目和汉密尔顿轻轨网</li>
<li>10公里以内GO火车票价3元</li>
<li>滑铁卢至多伦多全天双向GO火车运营</li>
<li>恢复Northlander火车服务</li>
</ul>
<p><strong><b>绿党</b></strong></p>
<ul>
<li>政府和个人购买电动车提供补助，积极扩大全省电动车基建</li>
<li>逐步废除内燃发动机到2050年实现碳中和</li>
</ul>
