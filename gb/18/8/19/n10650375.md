<tr><td VALIGN=TOP width="626"><h2 align=center>三星Galaxy Note 9与Note 8有何不同</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2018/08/GettyImages-1013915652-1-600x400.jpg" />
<h6>三星今年将上市的Galaxy Note 9。(TIMOTHY A. CLARY/AFP/Getty Images)
</h6>
<hr>
<p>【大纪元2018年08月19日讯】（大纪元记者林南编译报导）<ahref="/gb/tag/%E4%B8%89%E6%98%9F.md#1">三星</a>今年推出的新Galaxy Note 9与去年的Galaxy Note 8相比如何？</p>
<p>《商业内幕》总结说，Galaxy Note 9具有更大的电池，一些先进的相机功能和更多的启动存储。它还配备了一个更先进的S Pen，可用作遥控器。</p>
<p>Galaxy Note 9也更大、更重，这款新手机起价为1000美元，与iPhone X一样，是目前最昂贵的智能手机之一。</p>
<p>但Galaxy Note 9的整体设计与Galaxy Note 8非常相似，从硬件的角度来看，与去年的Note 8相同，包括无线快速充电，并具有超薄边框的近乎边缘到边缘的屏幕。</p>
<p>因此，您可以考虑从Galaxy Note 8进行升级，也可尝试留住Galaxy Note 8，毕竟它仍然是一款出色的手机，而且现在价格已经降低了，以750美元就可以买得到。去年Note 8的起价为930美元。</p>
<h2>存储空间</h2>
<p>Galaxy Note 9只有两个存储选项：128GB或512GB。而Galaxy Note 8提供三种存储选项：64GB，128GB或256GB。</p>
<p>Galaxy Note 9有512GB microSD卡的空间，这意味着您可以将手机的总存储空间提高到1TB。Galaxy Note 8也有一个microSD插槽，但只能容纳256GB的卡。</p>
<h2>屏幕更大</h2>
<p>Galaxy Note 9采用6.4英寸Amoled屏幕，与Galaxy Note 8的6.3英寸Amoled屏幕相比，更大一点。</p>
<h2>电池寿命更长</h2>
<p>Galaxy Note 9具有更大的电池，这意味着它最有可能比Galaxy Note 8具有更长的电池寿命。Galaxy Note 9拥有一个4,000mAh的怪兽电池。而Galaxy Note 8拥有更小的3,300-mAh电池。</p>
<h2>重量</h2>
<p>Galaxy Note 9的重量为201克，而Galaxy Note 8的重量为195克。</p>
<h2>升级的相机功能</h2>
<p>虽然这两款手机的相机规格几乎相同：800万像素前置摄像头，双1200万像素后置摄像头，但<ahref="/gb/tag/%E4%B8%89%E6%98%9F.md#1">三星</a>为Galaxy Note 9的相机增添了一些新功能。</p>
<p>Galaxy Note 9的双摄像头系统正在获得三星新的摄像头功能，称为场景优化器，它可识别您正在拍摄的物体，并自动调整设置以拍摄出最佳照片。</p>
<p>相机还具有探伤检验功能，可让您知道图像是否模糊，拍摄对象是否闭眼睛了，镜头是否有污迹，或是否有背光影响图像质量。</p>
<p>Galaxy Note 9还有一项新功能，三星还同时在Galaxy S9上添加了这项新功能：双光??圈镜头。三星表示，手机的镜头可以感知存在的光量，并在F光圈之间自动调节。</p>
<h2>颜色</h2>
<p>Galaxy Note 9在美国市场上有海洋蓝和薰衣草色，以及国际市场上将有铜色和黑色。Galaxy Note 8采用的是午夜黑色，深海蓝，兰花灰和枫金制成。</p>
<h2>S Pen</h2>
<p>在过去，三星设计的S Pen与设备的颜色相匹配。但是使用Galaxy Note 9，三星改变了蓝色设备附带的S Pen的颜色，它是亮黄色，当你在屏幕上书写时，它也会显示为亮黄色。</p>
<p>Galaxy Note 9附带的S Pen功能也更先进。新款S Pen具有低功耗蓝牙功能，可以作为遥控器使用，在拍照片、演示文稿或控制音乐时，用户无需靠近手机就能实现，S Pen支持最远是30英尺的距离。</p>
<h2>操作系统</h2>
<p>Galaxy Note 9配备了比Galaxy Note 8更新的Android版本Android 8.1 Oreo。Galaxy Note 8则运行Android 7.1.1 Nougat。有点令人惊讶的是，这款手机将没有使用周一发布的最新版Android，Android 9 Pie。</p>
<p>与Galaxy Note 8相同的是：新设备有一个耳机插孔，有线和无线快速充电。还有一个专用按钮Bixby，那是三星的智能助手。新设备的屏幕分辨率也是2960 x 1440。#</p>
