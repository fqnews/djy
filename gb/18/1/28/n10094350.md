<tr><td VALIGN=TOP width="626"><h2 align=center>家长孩子翘首期盼的中半岛双语幼儿园（上）</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2018/01/DSC_0151-600x400.jpg" />
<h6>中半岛幼儿园Golden State Preschool在中文课程中，学生学习写中文。（湾区连锁双语幼儿园学校Golden State Preschool提供）
</h6>
<hr>
<p>【大纪元2018年01月28日讯】(大纪元记者余亨之旧金山湾区报导)Intelligent Learning（Golden State Preschool的前身），是中半岛最大的中文课后班，5年来深受家长和孩子的喜爱。在家长极力的要求下，为了给更多孩子提供优质的早幼儿双语教育，Golden State Preschool在南旧金山、圣马刁（San Mateo）筹备已久的两个分校，将于2月1日开学。许多妈妈闻讯赶来，其中还有怀胎5个月的准妈妈，前来为肚里的宝宝报名。</p>
<p>Golden State Preschool，每年参加美国早幼教协会（NAEYC）的年度博览会，探讨最佳幼教模式，不惜成本买进最新的玩具；幼儿园老师专业年轻，发自内心的喜爱每一个孩子；让家长头疼无奈的孩子，入园一年后判若两人，令妈妈喜出望外……Golden State Preschool不愧是中半岛最令家长期盼的双语幼儿园。</p>
<h2>中半岛中文课后班：Irina Yang从兼职老师到校长</h2>
<p>在美国大学念书的Irina Yang，毕业后在一家银行工作，收入优厚。但是率性活泼、乐于交往的Irina，无法忍受银行工作的悠闲、乏味、封闭，辞去了银行工作。一个偶然的机会，她去了一家课后学校（After School）做兼职老师，教书法、国画。</p>
<figure id="attachment_10094398" style="width: 300px" class="wp-caption aligncenter"><ahref="https://i.epochtimes.com/assets/uploads/2018/01/02_Golden-State-Preschool_Irina-Yang.jpg"><img class="size-small wp-image-10094398" src="https://i.epochtimes.com/assets/uploads/2018/01/02_Golden-State-Preschool_Irina-Yang-300x411.jpg" alt="" width="300" b="411" /></a><figcaption class="wp-caption-text">耐心十足又多才多艺的Golden State Preschool杨校长Irina Yang。(中半岛双语幼儿园GSPGolden State Preschool提供)</figcaption></figure>
<p>通常专业课的老师上课时，需要一名助手维持课堂纪律。但是Irina上课生动有趣，孩子学得津津有味，课堂秩序好，根本无需助手。老校长很惊讶于她的才华，再三请她做全职老师。</p>
<p>作为全职老师，Irina不仅教画画、书法，还教手工、奥林匹克数学（逻辑数学）等有趣的科目，以及大年纪班的中文。年岁已大的校长越来越器重Irina，跟她合作越多，最后把学校的经营权交给了Irina。</p>
<h2><strong>中半岛双语幼儿园</strong>GSP<strong>：寓教于游戏中</strong></h2>
<p>被誉为“幼稚教育之父”的德国教育家福禄贝尔强调：“幼儿教育是由游戏来辅导，游戏是此时期发展中的最高阶段，是一种喜悦、自由、平静，且可以和第三者共享的活动。幼儿藉游戏发展其智力、体力和道德心。”</p>
<p>Golden State Preschool的数学教育，采用最新的LEGO Education STEAM模式，通过乐高玩具丰富多彩的玩法、百变的创意，寓教于乐，点燃学前儿童对科学、技术、工程、艺术和数学的天生好奇心。</p>
<h2><strong>双语幼儿园GSP：给孩子最好教育，最新玩具</strong></h2>
<p>美国早幼教协会（NAEYC）的年度博览会，是全球早幼教领域专业水平最高、规模最大的年度全球性交流平台。全球幼儿教育作家汇集一堂，探讨国际前沿教育理念，探索和发现最新最全的早期教育资源、玩具、教具设备和教案等。</p>
<p>Golden State Preschool校长Irina，每年参加NAEYC年度博览会，汲取精华的幼儿教育经验；不惜成本购买最新教具、玩具。比如十多美金一个的星巴克（Starbuck）彩绘杯，比电脑还贵的乐高玩具等。只要是好的、对孩子有益的东西，Irina都会去添购，学校为此每年更新许多教学与游戏的资源。</p>
<h2><strong>湾区连锁双语幼儿园GSP</strong>：<strong>阳光贴心的专业教师团队</strong></h2>
<p>Golden State Preschool的教师均为研究生毕业，多数是教育专业的研究生，拥有香港教育大学幼儿教育部颁发的ECE认证（Early Childhood Education）。她们年轻，发自内心的喜欢孩子。每一次学校的开放日（Open House），老师们都会跟前来的小朋友一起玩得很开心，临走时小孩们都依依不舍。</p>
<figure id="attachment_10094473" style="width: 600px" class="wp-caption aligncenter"><ahref="https://i.epochtimes.com/assets/uploads/2018/01/DSC_0124.jpg"><img class="size-large wp-image-10094473" src="https://i.epochtimes.com/assets/uploads/2018/01/DSC_0124-600x402.jpg" alt="" width="600" b="402" /></a><figcaption class="wp-caption-text"><ahref="/gb/tag/%E4%B8%AD%E5%8D%8A%E5%B2%9B%E5%B9%BC%E5%84%BF%E5%9B%AD.md#1">中半岛幼儿园</a>Golden State Preschool在中文课程中，学生学习写书法。（湾区连锁双语幼儿园学校Golden State Preschool提供）</figcaption></figure>
<p>老师们时常关注孩子的心情变化，学习成绩的进步或后退。当看到小孩哭时，年轻的女教师会蹲在地上与小孩同高度（让孩子感觉到是在沟通，不是批评他），让小孩回答：为什么哭？或者带着小孩去洗一下脸，让他平静一下情绪；或带他到另一个教室，转换一下环境……她们绝不会把小孩拎起来，放到一边，让孩子哭到不想哭，这种现象在Golden State Preschool是根本不会出现的。Irina校长认为这样的做法会增加孩子的恐惧感，对小孩的未来成长将留下阴影。</p>
<p>Golden State Preschool是湾区唯一给半职教师买全职保险（可看牙）的学校，老师都有带薪假期，因此这里的教师汰换率（Turn Over）很低，老师有时间充分地了解孩子。这里的一年级老师或幼儿园老师，能认识全校的孩子；Irina 校长自己就能能记住学校里每个孩子的名字。</p>
<p>但是教师们不叫孩子的真实姓名，而是叫他们“宝贝”、“宝贝喝水”、“宝贝洗手”。小孩在这最先学会的中文单词是宝贝。在这个学校老师的眼里，没有一个小孩是不好的。</p>
<h2>令妈妈头疼不已的孩子，判若两人</h2>
<p>5岁的涛涛非常调皮、任性、不听话。性格爽朗、热情的妈妈拿他一点办法也没有。妈妈把涛涛送到Golden State Preschool说：“杨校长，你要做好准备呀，这可是个很难管的孩子！”</p>
<p>涛涛确实好动，上课心不在焉，我行我素；别人做数学游戏时，他独自一人不知道在忙什么。Irina校长经过细致观察，发现涛涛的心思非常细腻，能真诚帮助别人、用心照顾别人，比如帮老师削铅笔，他都能很认真地完成。</p>
<p>学校老师们对涛涛采取“赏识教育”的方式，不断发现他的优点，不断地赞扬，令孩子做得越来越好。涛涛入幼儿园一年后，与先前判若两人，他的爸爸、妈妈欣喜地说：“这还是我们的孩子吗？！”（待续）</p>
<p>这么好的双语幼儿园，家长怎能不翘首以盼呢！如果您正在为孩子选择幼儿园而烦恼，不妨去体验一下吧！<br />
每个班级名额有限，预约排位请从速!</p>
