  <p>据法国广播电台报导，法国总统奥朗德在政府受害者事务部长，巴黎市长的陪同下走过2015年11月13日巴黎恐袭案每一个遭到袭击的地方，向在那里被<ahref="/gb/tag/%E6%81%90%E6%80%96%E5%88%86%E5%AD%90.md#1">恐怖分子</a>袭击丧生的遇难者默哀。法国官方为每一个受袭的地方树立一个尺寸不大的纪念碑，奥朗德在每一个碑前敬读遇难者的名字，并向遇难者默哀一分钟。</p>
  <p>去年的巴黎恐袭案一共造成130人死亡。奥朗德今天向巴黎恐袭案受害者致哀的活动，完全按照去年的今天<ahref="/gb/tag/%E6%81%90%E6%80%96%E5%88%86%E5%AD%90.md#1">恐怖分子</a>袭击巴黎所发生的恐怖袭击事件时间演进进行，首先前往巴黎北部近郊圣德尼省法兰西体育场凭吊。</p>
  <p>在巴黎市区的悼念，奥朗德今天主要在巴黎第10 和第11区驻留，不同的街区成为巴黎恐袭案主要的血染场所。奥朗德从巴黎编钟音以及小柬埔寨两家酒肆饭馆开始凭吊，恐怖袭击巴黎的第二个分队在此射杀了13位无辜者。</p>
  <p>奥朗德还抵达不远的王泉路致哀，该街区的好味啤酒与卡萨诺斯特拉两家饭店，遭到恐怖分子袭击，5人死于恐怖者枪下。</p>
  <p>随后奥朗德在不远的伏尔泰酒吧凭吊。两名无辜者在此受伤，恐怖分子布拉欣姆阿布德斯拉姆引爆绑在身上的炸药带。</p>
  <p>奥朗德驻足默哀的地方，接下来还有美好时代酒馆，那里是巴黎恐袭案遭到袭击的饭店酒吧遇难者最多的地方，20人丧生，17人受伤。</p>
  <p>法国总统奥朗德纪念巴黎恐袭案一周年仪式最后一站是巴塔克兰剧院。恐怖袭击巴黎的恐怖分子第3个分队冲进巴塔克兰剧院，向舞台上下开枪扫射，共造成90人死亡。</p>
  <p>袭击案发生一年后，法国各地纪念活动多天前就以不同方式出现。奥朗德官方纪念仪式后，所有的凭吊地都立即有许多巴黎人献花默哀，还有许多人从外地赶来参加纪念活动。</p>
  <div class="getty embed image" style="background-color: #fff; display: inline-block; font-family: 'Helvetica Neue',Helvetica,Arial,sans-serif; color: #a7a7a7; font-size: 11px; width: 100%; max-width: 594px;">  <div style="padding: 0; margin: 0; text-align: left;"><a style="color: #a7a7a7; text-decoration: none; font-weight: normal !important; border: none; display: inline-block;" href="http://www.gettyimages.com/detail/623006144" target="_blank">Embed from Getty Images</a></div>  <div style="overflow: hidden; position: relative; height: 0; padding: 66.666667% 0 0 0; width: 100%;"><a style="display: inline-block; position: absolute; top: 0; left: 0; width: 100%; height: 100%; margin: 0;" src="//embed.gettyimages.com/embed?assets=623006144,623006618,622998146,622997078,622996034&amp;et=TWbdrxEQTXR0lWs4vygX3w&amp;viewMoreLink=on&amp;sig=YFYEt8hOUF1R_YyJ_TmWfsbIR-JUQMgNJvv2nN7kcnQ=&amp;caption=true" width="594" b="396" frameborder="0" scrolling="no"></a></div>  <p style="margin: 0;">  </div>  <p>巴黎恐袭案受害者家属与幸存者组成的不同协会包括111315真相协会，都积极投入纪念活动。但他们都希望以尊重简约的形式向遇难者致敬。该协会还邀请巴黎人今天晚上在每一家的窗口点燃蜡烛，以纪念巴黎恐怖袭击案的遇难者。</p>
  <p>巴黎恐袭案一名幸存者表示，恐怖袭击事件阴影没有解除，随时都会在我们身边发生。但我们应当顽强活下去，就当没有恐怖威胁这回事。但我们不会忘记遇难者。</p>
  <p>佩里格居民森德瑞克：“带着我的孩子来很重要，向他们解释发生了什么，悼念所有受悲剧影响的受害者。”</p>
  <p>“那些令人焦虑的回忆还留在意识里”，住在巴黎郊区的居勒说，“我们现在对周遭环境更小心翼翼，不管是搭巴士、搭地铁、或走在路上，对声音变得很敏感，像是救护车、消防车的声音，让人压力很大、很紧张”。</p>
