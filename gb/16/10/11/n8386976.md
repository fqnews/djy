  <p>据中央社报导，<ahref="/gb/tag/%E4%B8%89%E6%98%9F.md#1">三星</a>公司发布声明说：“我们最近调整生产数量，以利彻底调查和品质管制，但因为将消费者安全列为最高优先，我们已作出停止生产Galaxy Note 7的最后决定。”</p>
  <p>南韩联合新闻通讯社（Yonhap）报导，正式宣布停产意味三星将不再生产这款机型。了解内情人士也向路透社透露，这显示三星将永久停售这种旗舰机型。</p>
  <p>彭博社报导，根据三星公司透过电子邮件发布的声明，三星将于10月13日至12月31日间为南韩的Note 7使用者退款，或者更换其它机型。</p>
  <p>三星电子决定停止销售和提供Galaxy Note 7换新服务后，今天股价终场重摔8.0%，最终收在154万韩元，市值蒸发188亿美元。</p>
  <figure id="attachment_8329759" style="width: 450px" class="wp-caption aligncenter"><ahref="https://i.epochtimes.com/assets/uploads/2016/09/1609230415131758.jpg"><img class="wp-image-8329759 size-medium" src="https://i.epochtimes.com/assets/uploads/2016/09/1609230415131758-450x323.jpg" width="450" b="323" /></a><figcaption class="wp-caption-text">全球传出数起Samsung Note 7燃烧事故，韩国手机大厂三星日前宣布，将针对全球Note 7用户提供更换新机服务，并暂停销售。台湾三星9月23日起开始进行Note 7换机作业，不少用户下午到销售门市更换手机。（中央社）</figcaption></figure>  <p>Note 7因发生多起充电起火意外，三星在接获客户投诉后，9月初在全球召回250万支Note 7。三星Note 7二度出包，伤了不少资深“星粉”的心，感叹“三星把自己一手好牌打坏了”、“要换的下一支手机变未知数”，也有网友无奈说，“时间成本回不去了”。</p>
  <p>分析师说，Note 7若永久停卖，三星损失恐将上看170亿美元，三星其它手机产品在消费者和电信商心中的形象也将受损。</p>
  <p>台湾三星公布在台回收配套方案，各大电信业者即日起开始受理退货，三星智慧馆预计于10月15日起开始受理，优惠换购即日起至11月15日止，可换购S7 edge、Note 5，另加码新台币3,000元现金礼券。</p>
  <p><strong>Note 7深陷电池爆炸危机 出包事件簿</strong></p>
  <p>三星电子8月抢在苹果iPhone 7推出前，发布主打防水防尘的Galaxy Note 7，上市两个多月经历自燃、召修与机上停用等风波，如今换机后再传爆炸，迫使三星宣布全球停售停换。以下是三星Note 7出包事件整理：</p>
  <p><strong>● Note 7风光上市</strong></p>
  <p>三星8月2日正式发表Galaxy Note 7，主打防水防尘、虹膜辨识功能，并于8月19日正式开卖，盼与苹果（Apple）的iPhone争抢市场大饼。</p>
  <p><strong>● 爆炸事件频传</strong></p>
  <p>Galaxy Note 7智慧型手机首传起火事件，一名使用者8月24日在南韩为手机充电时发生爆炸，之后陆续传出35起自燃与爆炸案例。三星解释是锂电池瑕疵。</p>
  <p><strong>● 全球召修</strong></p>
  <p>三星9月2日宣布将针对全球Note 7用户提供换机服务并暂停销售Note 7，总共召修250万支Note 7，并于9月19起提供Galaxy Note 7用户备用手机。</p>
  <p><strong>● 美航空总署建议机上勿用</strong></p>
  <p>美国联邦航空总署（FAA）9月8日强烈建议旅客，机上勿使用与托运三星智慧型手机Galaxy Note 7，全球多家航空业者跟进。</p>
  <p><strong>● 美国消费安全机构吁停用</strong></p>
  <p>美国消费品安全委员会9月9日示警说，Galaxy Note 7智慧型手机持有人应停止使用这种装置，因为电池有起火威胁。</p>
  <p><strong>● 三星吁用户停用</strong></p>
  <p>三星电子9月10日建议南韩和美国消费者“停用Galaxy Note 7手机”。</p>
  <p><strong>● 三星提名李在镕任董事</strong></p>
  <p>在三星面临多年来最严重的危机之际，三星电子9月12日提名病重的会长之子李在镕担任董事。</p>
  <p><strong>● 三星发布Note7软体更新</strong></p>
  <p>三星电子9月14日发布软体更新，将限制Galaxy Note 7智慧型手机南韩用户的电池只能充饱60%，以降低爆炸风险。</p>
  <p><strong>● 美国正式宣布召回</strong></p>
  <p>美国消费品安全委员会9月15日宣布，召回大约100万支三星电子公司生产的Galaxy Note 7智慧型手机，原因是这款手机发生多起电池起火事件。同日，美国运输部下令搭机乘客不得携带Galaxy Note 7手机登机，除非在航程中保持关机，而且不能为手机充电。</p>
  <p><strong>● 新版Note 7 美FAA解禁</strong></p>
  <p>三星在多国陆续执行Note 7换机作业后，南韩已于9月22日凌晨公布允许新版Note 7于机上使用，美国联邦航空总署9月26日也宣布，已对新版Note 7解禁。</p>
  <p><strong>● 换机后又着火</strong></p>
  <p>一支已更换过的三星Note 7手机，10月5日在从肯塔基州路易斯市飞往巴尔的摩的美国西南航空班机上冒烟，导致机上乘客紧急疏散。</p>
  <p><strong>● Note 7暂时中断生产</strong></p>
  <p>Galaxy Note 7持续引发安全隐忧，主要经销商停止供应<ahref="/gb/tag/%E9%80%80%E6%8D%A2%E8%B4%A7.md#1">退换货</a>后，三星电子10月10日宣布将“调整”Note 7产量，并称所谓“暂时调整生产进度”为暂时中断生产，并非停产。</p>
  <p><strong>● 全球停售停换</strong></p>
  <p>由于产品持续发生安全问题，三星电子10月11日宣布，全球停售、停换Galaxy Note 7智慧手机，并建议所有使用者立即停止使用这款手机。</p>
  <p><strong>● 三星股价大跌</strong></p>
  <p>彭博社报导，三星电子宣布停售停换Note 7消息后，盘中一度重挫5%。</p>
  <p><strong>● 分析师看法</strong></p>
  <p>三星电子要求零售商停售与停换Note 7智慧型手机后，分析师认为继续生产Note 7手机已无意义；此外，数百万名智慧型手机消费者可能变心，让苹果与Google坐收渔翁之利。#</p>
