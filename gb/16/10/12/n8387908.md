  <p>南韩三星电子今天宣布，全球停售、停换Galaxy Note 7智慧手机，并建议消费者立即关机停用。稍后三星又宣布，已作出永久停产这款手机的最后决定，意味这款旗舰级手机将永久下架。</p>
  <p>美国消费品安全委员会（U.S. Consumer ProductSafety Commission）也呼应三星声明，并表示将持续调查9月15日美国宣布正式召修以来至少5起过热或起火事件。</p>
  <p>以下是三星Note 7出包事件整理：</p>
  <p>● Note 7风光上市</p>
  <p>三星8月2日正式发表Galaxy Note 7，主打防水防尘、虹膜辨识功能，并于8月19日正式开卖，盼与苹果（Apple）的iPhone争抢市场大饼。</p>
  <p>● 爆炸事件频传</p>
  <p>Galaxy Note 7智慧型手机首传起火事件，一名使用者8月24日在南韩为手机充电时发生爆炸，之后陆续传出35起自燃与爆炸案例。三星解释是锂电池瑕疵。</p>
  <p>● 全球召修</p>
  <p>三星9月2日宣布将针对全球Note 7用户提供换机服务并暂停销售Note 7，总共召修250万支Note 7，并于9月19起提供Galaxy Note 7用户备用手机。</p>
  <p>● 美航空总署建议机上勿用</p>
  <p>美国联邦航空总署（FAA）9月8日强烈建议旅客，机上勿使用与托运三星智慧型手机Galaxy Note 7，全球多家航空业者跟进。</p>
  <p>● 美国消费安全机构吁停用</p>
  <p>美国消费品安全委员会9月9日示警说，Galaxy Note 7智慧型手机持有人应停止使用这种装置，因为电池有起火威胁。</p>
  <p>● 三星吁用户停用</p>
  <p>三星电子9月10日建议南韩和美国消费者“停用Galaxy Note 7手机”。</p>
  <p>● 三星提名李在镕任董事</p>
  <p>在三星面临多年来最严重的危机之际，三星电子9月12日提名病重的会长之子李在镕担任董事。</p>
  <p>● 三星发布Note7软体更新</p>
  <p>三星电子9月14日发布软体更新，将限制Galaxy Note 7智慧型手机南韩用户的电池只能充饱60%，以降低爆炸风险。</p>
  <p>● 美国正式宣布召回</p>
  <p>美国消费品安全委员会9月15日宣布，召回大约100万支三星电子公生产的Galaxy Note 7智慧型手机，原因是这款手机发生多起电池起火事件。同日，美国运输部下令搭机乘客不得携带Galaxy Note 7手机登机，除非在航程中保持关机，而且不能为手机充电。</p>
  <p>● 新版Note 7 美FAA解禁</p>
  <p>三星在多国陆续执行Note 7换机作业后，南韩已于9月22日凌晨公布允许新版Note 7于机上使用，美国联邦航空总署9月26日也宣布，已对新版Note 7解禁。</p>
  <p>● 换机后又着火</p>
  <p>一支已更换过的三星Note 7手机，10月5日在从肯塔基州路易斯市飞往巴尔的摩的美国西南航空班机上冒烟，导致机上乘客紧急疏散。</p>
  <p>● Note 7暂时中断生产</p>
  <p>Galaxy Note 7持续引发安全隐忧，主要经销商停止供应退换货后，三星电子10月10日宣布将“调整”Note 7产量，并称所谓“暂时调整生产进度”为暂时中断生产，并非停产。</p>
  <p>● 全球停售停换</p>
  <p>由于产品持续发生安全问题，三星电子10月11日宣布，全球停售、停换Galaxy Note 7智慧手机，并建议所有使用者立即停止使用这款手机。</p>
  <p>● 正式宣布永久下架</p>
  <p>三星在11日傍晚正式宣布，决定全球停售、停换并永久停止生产Note 7，也建议消费者别再使用这种装置，这款手机永久下架。</p>
  <p>● 三星股价大跌</p>
  <p>三星股价重挫8%，以154万韩元作收，昨天则惨跌1.5%。</p>
  <p>● 分析师看法</p>
  <p>分析师指出，智慧型手机市场已趋饱和，竞争比以往更激烈，三星可能为了对抗对手苹果公司（Apple）的新机iPhone 7，在匆忙下推出Note 7，结果以产品下架收场，使公司面临严重危机，品牌形象可能受到严重影响。</p>
  <p>●损失与影响</p>
  <p>分析师说，Note 7永久下架，三星电子损失恐上看170亿美元。财务分析师温莎（Richard Windsor）在客户报告中说：“这是我首度见识到一项产品回收可以搞得如此一团槽。至于这对三星品牌的伤害，将是史无前例。”</p>
  <p>策略分析公司（Strategy Analytics）分析师莫斯顿（Neil Mawston）说，三星旗舰机黯然下架，苹果和谷歌（Google）的新机将伺机取代这个缺口。苹果上月已发表最新iPhone 7，谷歌也将在本月稍晚推出Pixel新机。</p>
  <p>然而，莫斯顿说，Note 7停产，最大受益者可能是生产大量搭载谷歌Android平台高价手机的亚洲厂。 （转自中央社）</p>
