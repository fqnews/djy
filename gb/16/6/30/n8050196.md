  <p>美国人力资源服务机构Universum Global的一项最新调查，罗列出全球对毕业生最友好的企业名单，其中<ahref="/gb/tag/%E8%B0%B7%E6%AD%8C.md#1">谷歌</a>、苹果以及安永会计师事务所分居前三名。</p>
  <p>在上榜的前15名企业中，以顾问咨询公司占最大宗，共6家挤进排行，且多集中在世界金融中心纽约，它们分别是：安永、普华永道、德勤咨询、毕马威、麦肯锡以及波士顿咨询。</p>
  <p>其次是高科技产业，有谷歌、苹果与微软等3家入排行；其它还有金融（高盛、摩根大通与摩根士丹利）、化妆品的欧莱雅、民生消费的宝洁以及汽车制造BMW上榜。</p>
  <p>以下就是对毕业生最友好的企业名单。</p>
  <h4>1. 谷歌（Google）</h4>  <ul>  <li>产业：高科技</li>  <li>总部：加州山景城（Mountain View）</li>  </ul>  <h4>2. 苹果（Apple）</h4>  <ul>  <li>产业：高科技</li>  <li>总部：加州库柏蒂诺（Cupertino）</li>  </ul>  <h4>3. 安永（Ernst &amp; Young, EY）</h4>  <ul>  <li>产业：顾问公司</li>  <li>总部：英国伦敦</li>  </ul>  <h4>4. 高盛（Goldman Sachs）</h4>  <ul>  <li>产业：金融</li>  <li>总部：纽约</li>  </ul>  <h4>5. 普华永道（PricewaterhouseCoopers, PwC）</h4>  <ul>  <li>产业：顾问公司</li>  <li>总部：纽约</li>  </ul>  <h4>6. 德勤咨询（Deloitte）</h4>  <ul>  <li>产业：顾问公司</li>  <li>总部：纽约</li>  </ul>  <h4>7. 微软（Microsoft）</h4>  <ul>  <li>产业：高科技</li>  <li>总部：华盛顿州雷德蒙（Redmond）</li>  </ul>  <h4>8. 毕马威（KPMG）</h4>  <ul>  <li>产业：顾问公司</li>  <li>总部：荷兰阿姆斯特尔芬（Amstelveen）</li>  </ul>  <h4>9. 欧莱雅（L&#8217;Oréal）</h4>  <ul>  <li>产业：化妆品</li>  <li>总部：法国</li>  </ul>  <h4>10. 摩根大通（JPMorgan Chase）</h4>  <ul>  <li>产业：金融</li>  <li>总部：纽约</li>  </ul>  <h4>11. 麦肯锡（McKinsey &amp; Co.）</h4>  <ul>  <li>产业：顾问公司</li>  <li>总部：纽约</li>  </ul>  <h4>12. 波士顿咨询（Boston Consulting Group）</h4>  <ul>  <li>产业：顾问公司</li>  <li>总部：马萨诸塞州波士顿</li>  </ul>  <h4>13. 宝洁（又译：宝碱，Procter &amp; Gamble）</h4>  <ul>  <li>产业：民生消费品</li>  <li>总部：俄亥俄州辛辛那提</li>  </ul>  <h4>14. 摩根士丹利（Morgan Stanley）</h4>  <ul>  <li>产业：金融</li>  <li>总部：纽约</li>  </ul>  <h4>15.
