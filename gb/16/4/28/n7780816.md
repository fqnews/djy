  <p>据澳新社（AAP）报导，<ahref="/gb/tag/%E7%BB%B4%E7%9C%81.md#1">维省</a>工党政府在4月27日宣布2016/17财政年度<ahref="/gb/tag/%E9%A2%84%E7%AE%97.md#1">预算</a>案，预算案预测2016/17年度实现29亿澳元财政盈余，政府在预算案中承诺对基础设施投资，维持AAA的信用级别。政府宣称在全澳范围内，维省增长步伐最快。</p>
  <p>维省财政厅长帕拉斯（Tim Pallas）说：“我不希望（预算案）听起来过于乐观，但这个预算案确实显示，维省经济是多么稳健，我们是多么严肃地履行管理财政的责任。”</p>
  <p>在2018年11月份大选前，维省财政状况将一直维持稳健的盈余。财政厅预测2017/18、2018/19、以及2019/20财政年度的盈余额将分别达到18亿、21亿和25亿。在未来四个财政年度内，财政盈余将累积93亿澳元，比政府上次预测高27亿澳元。</p>
  <p>帕拉斯说：“预算案将重点支持学校、医院和公共交通，同时也重视为未来积累资金。”</p>
  <p>尽管财政收入骤增，但政府也在一些领域调高税率，例如外国人购买住宅的印花税率从3%调高到7%﹔对外国人购买但空置的住宅征收1.5%土地税（此前税率为0.5%）﹔褐煤企业矿产资源税率被调高至三倍。这些措施将在未来四年内增加7亿澳元收入。</p>
  <p>财政债务占经济总量的比率在2016/17和2017/18年度将分别达到4.7%和4.8%，但帕拉斯预测该比率可能上涨到6.2%，达到工党刚执政时的水平，因为工党政府计划通过举债来资助未来主要项目。</p>
  <p><strong>创造就业</strong>：预算案提供2.37亿澳元吸引投资者到维省投资，引进高端人才，以及吸引大型跨国企业到维省建立业务。</p>
  <ul>  <li>1.16亿澳元吸引投资项目和辅助项目</li>  </ul>  <ul>  <li>6600万澳元扩充并支持维省政府的“商业办公室”网路项目</li>  </ul>  <ul>  <li>3400万澳元用于向世界宣传维省</li>  </ul>  <ul>  <li>1200万澳元加强“维省企业参与政策”（VIPP）</li>  </ul>  <ul>  <li>900万澳元资助“商业活动基金”</li>  </ul>  <ul>  <li>5000万澳元资助“省长的就业与投资基金”帮助企业成长并吸引投资</li>  </ul>  <p><strong>公共交通和道路</strong>：预算案提供19亿澳元改善城市交通系统。</p>
  <p>墨尔本公共交通将获得以下资助：</p>
  <ul>  <li>8.75亿澳元为Metro隧道项目订购28辆高容量列车</li>  </ul>  <ul>  <li>1.05亿澳元购买5辆X&#8217;Trapolis列车</li>  </ul>  <ul>  <li>5.88亿澳元把南Morang铁路线延伸至Mernda</li>  </ul>  <ul>  <li>1.4亿澳元改善Hurstbridge铁路线（包括Heidelberg与Rosanna之间的双线工程）</li>  </ul>  <ul>  <li>1.34亿澳元改善市中心环城铁路的火警与安全监控系统</li>  </ul>  <ul>  <li>5000万澳元更新弗兰克斯顿（Frankston）火车站</li>  </ul>  <p>郊区铁路系统获得13亿澳元资助：</p>
  <ul>  <li>5.18亿澳元更新巴拉瑞特（Ballarat）铁路线</li>  </ul>  <ul>  <li>1.41亿澳元用于郊区铁路线路维护</li>  </ul>  <ul>  <li>1.98亿澳元资助V/Line</li>  </ul>  <ul>  <li>超过1.86亿澳元用于额外服务、铁轨翻新和车站翻新</li>  </ul>  <ul>  <li>2.8亿澳元购买新的VLocity列车车厢</li>  </ul>  <p>城市道路：</p>
  <ul>  <li>4.37亿澳元兑现大选承诺，改建波尔顿（Bolton）区Thompsons路、Yan Yean路、O&#8217;Herns路和Hoddle街</li>  </ul>  <ul>  <li>2.28亿澳元改建Dohertys路与Hallam路的十字路口，拓宽Plenty路</li>  </ul>  <ul>  <li>1000万澳元改善远郊道路</li>  </ul>  <p><strong>教育</strong>：政府出资9.24亿澳元修建更多的教室和学校</p>
  <ul>  <li>3.85亿澳元改建学校</li>  </ul>  <ul>  <li>2.87亿澳元建新学校，购买土地</li>  </ul>  <ul>  <li>9200万澳元在巴拉瑞特（Ballarat）、班尤勒（Banyule）、班迪戈（Bendigo）、凯西（Casey）、吉朗（Geelong）、吉普斯兰（Gippsland）、莫纳什（Monash）、威特利西（Whittlesea）、温德姆（Wyndham）和雅拉山脉（Yarra Ranges）地区建技术学校</li>  </ul>  <ul>  <li>6360万澳元用于可移动教室</li>  </ul>  <ul>  <li>5000万澳元在新区修建社区教育中心</li>  </ul>  <ul>  <li>增加2800万澳元拆除校内石棉</li>  </ul>  <ul>  <li>出资1800万澳元在偏远地区的100所中学修建专用门诊室</li>  </ul>  <p>预算案还向健康领域资助29亿澳元，确保维省居民能够得到健康护理。</p>
