  <p>李进答：Joy您好，方向盘启动时会锁紧，原因可能是连锁装置（Interlocking Mechanism）年久需要润滑一下。<br />  您所指出的“灯”是引擎故障灯Check Engine Light吗？如果是，需要请修车人士取读故障码，然后根据故障码才能具体判断故障原因，不一定都是油箱盖的问题。</p>
  <p>读者Joy续问：李先生您好，谢谢您在百忙之中还回信给我。我曾因为方向盘锁紧，到修车厂去寻求解决。修车厂的人说，可能是我曾在2010年冬天时在美国东北部开车几个月，下雪的道路上撒盐巴以致让机械脏了，因此更换了一些器具，但修车厂的人说不能保证方向盘就不会锁紧。而我花了一笔不小的钱后，方向盘偶而还是会锁紧，需要转几次才会正常。<br />  而亮灯的部分，两个修车场都采取更换加油盖处理，灯暂时熄灭，但几天后亮灯依旧。上个月那个修车场告诉我，是Emission的问题，需要一个机器喷（因为和汽油挥发有关系），只能到原车场去，因为该修车厂没有该工具。而我刚搬到亚特兰大，亚城规定需要检查汽车的Emission。</p>
  <p>李进答：Joy您好，方向盘锁紧的原因看来修车厂还是没有正确的判断故障所在，也没有完全修理好。感觉上需要进一步检查前悬挂系统的连杆和轴承是否有因路面的坑或不小心碰到马路路边有损伤？如果问题不大先开着再说。<br />  引擎故障灯Check Engine Light亮灯首先应噶检查故障码，而不是随便更换油箱盖。也不是用机器喷就解决。您需要去另一家修理厂请他们进一步诊断才行。拿到故障码之后才能进一步做判断。不要轻易更换东西。</p>
  <p>读者Joy问：李先生您好，好，我会依照您说的去找另一个修车场找出引擎灯号的故障码来。请问故障码的英文是Error code of engine light吗？再次谢谢您在百忙之中回信帮忙解决我的汽车问题，我感激不尽。</p>
  <p>李进答：Joy您好，故障码的英文是OBD error code (On Board Diagnostic Error Code)。多数是四位数字。<br />  <ahref="http://www.obd-codes.com/trouble_codes/">http://www.obd-codes.com/trouble_codes/</a></p>
  <p>李进<br />  具三十余年汽车维修保养经验，对汽车买卖也很熟悉。<br />  如果您有汽车维修与保养方面的问题，欢迎发信至<ahref="mailto:car_qna@epochtimes.com">car_qna@epochtimes.com</a>，本报将转发李进解答。来信请注明汽车型号、牌子及年份。◇</p>
