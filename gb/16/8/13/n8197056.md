  <p>6）迈克尔·<ahref="/gb/tag/%E8%8F%B2%E5%B0%94%E6%99%AE%E6%96%AF.md#1">菲尔普斯</a>：5500万美元</p>
  <p>美国奥运选手不是国家养大的，但谁说游泳得不到报酬？<ahref="/gb/tag/%E8%8F%B2%E5%B0%94%E6%99%AE%E6%96%AF.md#1">菲尔普斯</a>（Michael Phelps）的收入就高达5500万美元。</p>
  <div class="getty embed image" style="background-color:#fff;display:inline-block;font-family:'Helvetica Neue',Helvetica,Arial,sans-serif;color:#a7a7a7;font-size:11px;width:100%;max-width:594px;">  <div style="padding:0;margin:0;text-align:left;"><ahref="http://www.gettyimages.com/detail/589032688" target="_blank" style="color:#a7a7a7;text-decoration:none;font-weight:normal !important;border:none;display:inline-block;">Embed from Getty Images</a></div>  <div style="overflow:hidden;position:relative;height:0;padding:66.666667% 0 0 0;width:100%;"><a src="//embed.gettyimages.com/embed/589032688?et=wP6JOOS2TBJwDMK6gEQYhg&#038;viewMoreLink=on&#038;sig=0YqFrd46Esd1VnZQJruSM4Q6s3z4uHdKnYByCcHmQW0=&#038;caption=true" width="594" b="396" scrolling="no" frameborder="0" style="display:inline-block;position:absolute;top:0;left:0;width:100%;height:100%;margin:0;"></a></div>  <p style="margin:0;">  </div>  <p>菲尔普斯最初成为众人关注的焦点是在2004年雅典奥运会上，他拿到6枚金牌，2枚铜牌，因此成为一个家喻户晓的人物。四年之后他从北京奥运会捧回8枚金牌。在2012年的伦敦奥运会上，他的表现被认为是“令人失望”的，菲尔普斯只赢得4金2银。时间来到了2016年，菲尔普斯再次回到巅峰状态。截至8月12日，他在里约的前四项游泳比赛中已斩获了4枚金牌，这使他的奥运金牌总数累积至22枚。</p>
  <p>他所代言的品牌包括安德阿默（Under Armour）公司，史必（Speedo），维萨(Visa)， 赛百味(Subway)，惠普(HP)，动力巴（Power Bar）和希尔顿酒店等。</p>
  <p>5）安迪·穆雷：8500万美元</p>
  <div class="getty embed image" style="background-color:#fff;display:inline-block;font-family:'Helvetica Neue',Helvetica,Arial,sans-serif;color:#a7a7a7;font-size:11px;width:100%;max-width:594px;">  <div style="padding:0;margin:0;text-align:left;"><ahref="http://www.gettyimages.com/detail/588947090" target="_blank" style="color:#a7a7a7;text-decoration:none;font-weight:normal !important;border:none;display:inline-block;">Embed from Getty Images</a></div>  <div style="overflow:hidden;position:relative;height:0;padding:66.666667% 0 0 0;width:100%;"><a src="//embed.gettyimages.com/embed/588947090?et=5GzsN4qsTxNTrzLm3MXydg&#038;viewMoreLink=on&#038;sig=NUI5MfWiENgPiqEo4FwcJSrxmu-z8Mglm9THufIU3N0=&#038;caption=true" width="594" b="396" scrolling="no" frameborder="0" style="display:inline-block;position:absolute;top:0;left:0;width:100%;height:100%;margin:0;"></a></div>  <p style="margin:0;">  </div>  <p>英国网球名将穆雷（Andy Murray）是三个大满贯冠军得主，穆雷于8月12日在里约顺利进入半决赛。在2012伦敦奥运会上，他击败了费德勒，将金牌带回家。穆雷的代言活动估计可赚5000万美元，他从职业网球赛中额外可获2300多万美元的收入。</p>
  <p>4）凯文·杜兰特：1亿美元</p>
  <p>27岁的美国篮球明星杜兰特（Kevin Durant）最近成为自由球员，现效力金州勇士队。杜兰特加入男篮美国队，这支队伍在<ahref="/gb/tag/%E9%87%8C%E7%BA%A6%E5%A5%A5%E8%BF%90.md#1">里约奥运</a>比赛中是夺冠大热门。</p>
  <div class="getty embed image" style="background-color:#fff;display:inline-block;font-family:'Helvetica Neue',Helvetica,Arial,sans-serif;color:#a7a7a7;font-size:11px;width:100%;max-width:594px;">  <div style="padding:0;margin:0;text-align:left;"><ahref="http://www.gettyimages.com/detail/588297678" target="_blank" style="color:#a7a7a7;text-decoration:none;font-weight:normal !important;border:none;display:inline-block;">Embed from Getty Images</a></div>  <div style="overflow:hidden;position:relative;height:0;padding:69.360269% 0 0 0;width:100%;"><a src="//embed.gettyimages.com/embed/588297678?et=iQsMhhOJQxhlG1iDjYxgCg&#038;viewMoreLink=on&#038;sig=KoghH_soga_zHPBRzBUXWTU2xLtogyN-pKuECWlyl30=&#038;caption=true" width="594" b="412" scrolling="no" frameborder="0" style="display:inline-block;position:absolute;top:0;left:0;width:100%;height:100%;margin:0;"></a></div>  <p style="margin:0;">  </div>  <p>杜兰特在2007年NBA选秀首轮以第二顺位加入了西雅图超音速（2008年更名为俄克拉何马城雷霆），荣荐年度最佳新秀。2009-10年赛季，21岁的杜兰特成为NBA史上最年轻的得分王。2013-14年赛季，杜兰特首次获得NBA最有价值球员奖。他代表美国国家队出战2010年世锦赛和2012年奥运会，成功两夺金牌，并成为2010年世锦赛的最有价值球员。</p>
  <p>2010年俄克拉何马城雷霆以8600万美元与他签下5年合约。然后，他与金州勇士队签下2年的合同。他的广告代言为他再添3600万美元收入。</p>
  <p>3）诺瓦克·德约科维奇：1.3亿美元</p>
  <div class="getty embed image" style="background-color:#fff;display:inline-block;font-family:'Helvetica Neue',Helvetica,Arial,sans-serif;color:#a7a7a7;font-size:11px;width:100%;max-width:594px;">  <div style="padding:0;margin:0;text-align:left;"><ahref="http://www.gettyimages.com/detail/587148446" target="_blank" style="color:#a7a7a7;text-decoration:none;font-weight:normal !important;border:none;display:inline-block;">Embed from Getty Images</a></div>  <div style="overflow:hidden;position:relative;height:0;padding:66.666667% 0 0 0;width:100%;"><a src="//embed.gettyimages.com/embed/587148446?et=JFFHA6z0T5p57nTk2-Xf5Q&#038;viewMoreLink=on&#038;sig=6OATaC6a4LONghdoiu6qsrLrCC2U88_j0xJEy5jsf8U=&#038;caption=true" width="594" b="396" scrolling="no" frameborder="0" style="display:inline-block;position:absolute;top:0;left:0;width:100%;height:100%;margin:0;"></a></div>  <p style="margin:0;">  </div>  <p>德约科维奇（Novak Djokovic）是塞尔维亚出生的网球明星。今年29岁的他是当今世界上单打头号选手。在他的职业生涯中，他已经赢得了惊人的12个大满贯赛冠军。里约奥运会是德约科维奇第三次参加的奥运比赛，但已经提前被淘汰出局，无缘里约奖牌。今年他在澳网和法网都拿到大满贯个人冠军，但奥运成绩不佳，他的最好奥运成绩是在2008年奥运会上赢得一枚铜牌。</p>
  <p>德约科维奇的收入是1.3亿美元。</p>
  <p>2）塞雷娜·威廉姆斯：1.35亿美元</p>
  <div class="getty embed image" style="background-color:#fff;display:inline-block;font-family:'Helvetica Neue',Helvetica,Arial,sans-serif;color:#a7a7a7;font-size:11px;width:100%;max-width:594px;">  <div style="padding:0;margin:0;text-align:left;"><ahref="http://www.gettyimages.com/detail/587470368" target="_blank" style="color:#a7a7a7;text-decoration:none;font-weight:normal !important;border:none;display:inline-block;">Embed from Getty Images</a></div>  <div style="overflow:hidden;position:relative;height:0;padding:67.676768% 0 0 0;width:100%;"><a src="//embed.gettyimages.com/embed/587470368?et=KskKrVR4SRd_rRoSc_PAmg&#038;viewMoreLink=on&#038;sig=Wd5e7RC-1ufkUgFhvw8NF1i96lEC1Ny9QHQYBNU1_W0=&#038;caption=true" width="594" b="402" scrolling="no" frameborder="0" style="display:inline-block;position:absolute;top:0;left:0;width:100%;height:100%;margin:0;"></a></div>  <p style="margin:0;">  </div>  <p>塞雷娜·威廉姆斯（Serena Williams）也许是网坛上最有统治力的女运动员，她已垄断了这项运动，赢得36个主要冠军头衔：22次单打冠军和14次双打冠军。人们习惯于称她为“小威”、“威女皇”。</p>
  <p>2012年伦敦奥运会上，塞雷娜·威廉姆斯获得网球女子单打金牌，同时也获得网球女子双打金牌，成为“双金王”，她更是史上唯一同时拥有单打生涯金满贯和双打生涯金满贯的网球选手。</p>
  <p>今年世界单打排名第一的小威也在里约奥运会上提前出局。</p>
  <p>她的职业生涯总奖金估计有8100万美元。在2015年，她在广告代言给她带来2000万美元的额外收入。</p>
  <p>1) 内马尔：2亿2500美元</p>
  <div class="getty embed image" style="background-color:#fff;display:inline-block;font-family:'Helvetica Neue',Helvetica,Arial,sans-serif;color:#a7a7a7;font-size:11px;width:100%;max-width:594px;">  <div style="padding:0;margin:0;text-align:left;"><ahref="http://www.gettyimages.com/detail/588325530" target="_blank" style="color:#a7a7a7;text-decoration:none;font-weight:normal !important;border:none;display:inline-block;">Embed from Getty Images</a></div>  <div style="overflow:hidden;position:relative;height:0;padding:63.973064% 0 0 0;width:100%;"><a src="//embed.gettyimages.com/embed/588325530?et=NIZwo1LDSo1RCJ05nQEo7g&#038;viewMoreLink=on&#038;sig=ABeoj_9IqE12vAeMrbV3CHJl5XM909rZMRROfysbkGg=&#038;caption=true" width="594" b="380" scrolling="no" frameborder="0" style="display:inline-block;position:absolute;top:0;left:0;width:100%;height:100%;margin:0;"></a></div>  <p style="margin:0;">  </div>  <p>小内马尔·达席尔瓦·桑托斯（葡萄牙语：Neymar da Silva Santos Júnior），简称内马尔，是巴西足球运动员，现时效力西甲球队巴塞罗那。</p>
  <p>内马尔在19岁时赢得了2011年南美足球先生的称号，且隔年2012年再次赢得此殊荣，2011年内马尔收到国际足联金球奖提名，获选第10名，他还赢得2011年国际足联普斯卡什奖的年度最佳进球奖。在巴塞罗那与梅西及苏亚雷斯组成“MSN组合”。 2014-2015赛季，“MSN组合”总共攻入122球，公认为史上最为强力的锋线组合。协助巴塞罗那成为史上第一支两夺三冠王的球队。</p>
  <p>巴西队在世界杯足球赛中表现不佳，这次在本土亮相奥运会，内马尔能否为巴西队带来突破，成为世界瞩目的焦点。</p>
  <p>而今夏刚同巴萨完成续约的内马尔的<ahref="/gb/tag/%E8%BA%AB%E4%BB%B7.md#1">身价</a>则是一路飙升，上期内马尔的身价只有1亿5200万，而在本期公布的榜单中，内马尔的身价已经十分接近梅西，达到2亿150万欧元（约2亿2500美元），目前位居足坛第二位。</p>
