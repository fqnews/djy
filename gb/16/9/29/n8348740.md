<tr><td VALIGN=TOP width="626"><h2 align=center>LINE布局聊天机器人 广邀开发者</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2019/12/c369fca47693132d48911e39739a0252-320x200.jpg" />
<h6></h6>
<hr>
	<p>【大纪元2016年09月29日讯】LINE大动作布局<ahref="/gb/tag/%E8%81%8A%E5%A4%A9.md#1">聊天</a>机器人技术与开放平台，并为全球开发者推出全新Messaging API，同时举办“LINE聊天机器人大赏”，最高奖金1000万日圆。</p>
<p>LINE今天在日本东京的Hikarie Hall举办LINEDEVELOPER DAY 2016，并于会中宣布LINE即时通讯软体中“全新<ahref="/gb/tag/%E8%81%8A%E5%A4%A9.md#1">聊天</a>机器人”的技术与开放平台策略。</p>
<p>LINE今年4月7日开始提供免费“LINE BOT API 试用”账号（LINE BOT API Trial Account）申请，让任何人都可以在LINE的平台上开发聊天机器人的多元应用。</p>
<p>LINE表示，至目前为止，全球的开发合作伙伴已利用将近2万个“LINE BOT API 试用”账号开发出各种LINE聊天机器人服务。</p>
<p>为鼓励更多业界伙伴开发聊天机器人，并吸引全球更多用户使用，LINE于LINE DEVELOPER DAY 2016发表一连串的相关计划，LINE将持续推动技术知识的分享，让外部开发者和企业共享LINE自由与开放的开发环境。</p>
<p>LINE于LINE DEVELOPER DAY 2016宣布推出可以使用全新Messaging API功能的开发者账号，拥有更强的聊天机器人功能及更好的开发条件，给予开发者更完善的支援及发挥空间。</p>
<p>LINE指出，全新Messaging API包含三种讯息类型，包括确认型（confirm type）：能利用简单的二择一进行回应，如是/要（yes）或不是/不要（No）；按键型（button type）：结合图片、文字及其他触动按钮；横向卷轴型（carousel type）：多个按键型讯息能以横向卷动的方式呈现，使多元的资讯能更直觉排列。</p>
<p>此外， LINE宣布将举办“LINE聊天机器人大赏”（LINE BOT AWARDS），将选出善用聊天机器人工具、提供最佳使用者经验并赋予全新价值的开发者。不论是个人或公司皆可参赛，LINE将提供最高1000万日圆的奖金给最终优胜者，并将有额外奖项给其他竞赛类别。(转自中央社)</p>
