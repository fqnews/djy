  <p>心情不好，压力太大，想找乐子，喜欢刺激……好吧各位，是时候把鞋带系得再紧一点，女生们放弃裙子吧，还有眼镜、首饰、假牙、假发、拖鞋什么的，通通放进包里就不要再带了，因为接下来我们要讲的是英国十五个最受欢迎的<ahref="/gb/tag/%E8%BF%87%E5%B1%B1%E8%BD%A6.md#1">过山车</a>！当然啦，游乐场也并不是靠刺激就能拿满分，让小伙伴们都玩得开心才是重点。知道你们各有所爱，以下排名并不分先后哦。那么，我们就一起来开启“飞行模式”吧！</p>
  <p><strong>?</strong><strong>Oblivion</strong></p>
  <p>这是世界上速度最快的垂直降落<ahref="/gb/tag/%E8%BF%87%E5%B1%B1%E8%BD%A6.md#1">过山车</a>，当然不是跳楼那么简单的事，那是真真的从60米高空急速“掉进”下面烟雾缭绕的黑洞里！对，就是这么简单粗暴的掉进去了。总共也就几秒钟的事情，但是估计大部分人都是重新回到地面的时候才来得及叫出声吧。Oblivion就是那种不论你玩过多少次永远都不会感到无聊的极限刺激，胆大的人自己去试喽。</p>
  <ul>  <li>? ● ?地点：Alton Towers</li>  <li>? ● ?高度：60米</li>  <li>? ● ?最高时速：110千米/小时</li>  </ul>  <p><strong>Infusion</strong></p>
  <p>这个蓝色轨道是世界上第一个完全建在水面上的过山车，两人并排的座位头顶吊在轨道下面，所以最开始的俯冲会让你在后面的“旅行”中被强大的惯性甩来甩去！在水面上俯冲、旋转，在铁架子之间穿来穿去，空中还有五个圈等着你去转。从脚下的地板被撤下开始，屏住呼吸起飞啦！</p>
  <ul>  <li>? ● ?地点：Blackpool Pleasure Beach</li>  <li>? ● ?高度：33米</li>  <li>? ● ?最高时速：80千米/小时</li>  </ul>  <p><strong>Galactia</strong></p>
  <p>世界上第一个模拟飞行的过山车，也就是过去的Air，去年重整旗鼓后才改的新名字。不要小看这个薄荷绿的小清新哦，它有一段螺旋式的神奇轨道，再加上座椅的天才设计使游戏者的后背吊在轨道上，也就是说当撤掉脚下踏板，座椅带着你面朝下这么一转——还没启动就有了飞翔的感觉，超人也就是这样了吧，路过照相机记得摆好pose哦！飞完就一个字：“爽！”</p>
  <ul>  <li>? ● ?地点：Alton Towers</li>  <li>? ● ?高度：20米</li>  <li>? ● ?最高时速：75千米/小时</li>  </ul>  <p><strong>Nemesis Inferno</strong></p>
  <p>又是一个在轨道下面吊着飞的选手，红色的轨道有很多扭转再加上高速和惯性，最后的效果就像置身于一个失重的滚筒中。玩家们会围绕着一座喷发的火山飞窜，不过大家可能顾不上担心火山的事情吧。</p>
  <ul>  <li>? ● ?地点：Thorpe Park</li>  <li>? ● ?高度：29?米</li>  <li>? ● ?最高时速：80千米/小时</li>  </ul>  <p><strong>Nemesis</strong></p>

  <ul>  <li>? ● ?地点：Alton Towers</li>  <li>? ● ?高度：13米</li>  <li>? ● ?最高时速：80千米/小时</li>  </ul>  <p><strong>?</strong><strong>Stealth</strong></p>

  <ul>  <li>? ● ?地点：Thorpe Park</li>  <li>? ● ?高度：62米</li>  <li>? ● ?最高时速：130千米/小时</li>  </ul>  <p><strong>?</strong><strong>G-Force</strong></p>

  <ul>  <li>? ● ?地点：Thorpe Park</li>  <li>? ● ?高度：39米</li>  <li>? ● ?最高时速：92千米/小时</li>  </ul>  <p><strong>Thirteen</strong></p>
