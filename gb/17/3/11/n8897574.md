<tr><td VALIGN=TOP width="626"><h2 align=center>百年一遇暴雨 反致奥克兰自来水荒</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2020/07/0484f39860f2616dc9fa121b2e5587f0-320x200.png" />
<h6></h6>
<hr>
	<p>【大纪元2017年03月11日讯】（大纪元记者韩香茗综合报导）因遭遇百年一遇的<ahref="/gb/tag/%E6%9A%B4%E9%9B%A8.md#1">暴雨</a>袭击，<ahref="/gb/tag/%E5%A5%A5%E5%85%8B%E5%85%B0.md#1">奥克兰</a>自来水处理能力大大降低，致使该市面临23年不遇的水荒。自来水公司警告说，如市民不减少用水，可能因供应不足而不得不调用非完全处理水，但那样的话，水质将无法达到饮用水标准，市民将不得不煮水。</p>
<p>3月10日，<ahref="/gb/tag/%E5%A5%A5%E5%85%8B%E5%85%B0.md#1">奥克兰</a>最大自来水公司（Watercare）向民众公布了自来水供应情况，同时敦促市民在本周的<ahref="/gb/tag/%E6%9A%B4%E9%9B%A8.md#1">暴雨</a>过后，在3月底前每人每天减少20升用水量；不然的话，该公司将不得不发布把水煮沸的通知。</p>
<p>该公司表示，这是1994年（干旱）以来，<ahref="/gb/tag/%E6%96%B0%E8%A5%BF%E5%85%B0.md#1">新西兰</a>最大城市首次限水。正常情况下，奥克兰人平均每人每天消耗160升水，全市每天耗水量约为4.5亿升。自周四（8日）宣布节水后，用水量已经降至4.2亿升（减少了3000万升），但还需减少更多。目前设定的节水目标为每天4亿升，即每天减少5000万升。</p>
<p><strong>暴雨致水处理能力减半</strong></p>
<p>本周的暴雨使Watercare公司的水处理能力降低了50%，原因是集水区注入了大量带有泥浆的水，使原水非常浑浊，看上去跟橘子水一样，大大增加了处理难度。不过，自来水公司表示，自来水质量并未因此而降低。</p>
<p>该公司发言人马克·伯恩（Mark Bourne）说，水质并未受到影响，但问题在于集水量，不到24小时接收了两个月的降水量。这场暴雨非常罕见，自来水供应从未受到过这样的影响，他认为科学家和水分析师将最终证实目前的暴雨有多罕见。</p>
<p>他说：“我在自来水行业已经工作了27年，从未遇到过这样的情况。其它4个水处理厂的生产量也已达到最高水平。”</p>
<p>Watercare公司首席执行官拉文·杰督拉姆（Raveen Jaduram）表示，要想使水坝中的水降下去，需要持续一段时间的干燥天气，之后Ardmore处理厂才能恢复正常生产。</p>
<p>杰督拉姆说：“我们在努力确保为消费者提供可直接饮用的高品质自来水，目前可以做到，但为了保持这种质量，我们已经不得不减少了处理的水量。”不过，伯恩表示，水短缺不会影响到水价。</p>
<p><strong>医疗官：自来水仍可安全饮用</strong></p>
<p>据Watercare公司健康医疗官大卫·辛克莱尔（David Sinclair）证实，目前的自来水仍然可以安全饮用。通常情况下，水处理厂将原水净化1万倍，目前净化了1千倍，仍然是高品质的饮用水。他解释说，即使引入非完全处理水，致病的风险也是相当低的，经过煮沸的话，可以有效地预防疾病。</p>
<p><strong>市长：若暴雨持续</strong> <strong>将考虑新措施</strong></p>
<p>3月10日，奥克兰市长费尔·高夫（Phil Goff）在首都惠灵顿结束与总理英格利希（Bill English）的会晤后，立即返回奥克兰，并赶往Ardmore处理厂察看水情。</p>
<p>高夫说，如果今晚的降雨造成更大压力的话，就要重新考虑Ardmore处理厂的水处理方式。本周的暴雨可能是百年一遇的，但如果因气候变化，这种情况变得更加常见的话，就需要采取新的措施了。</p>
<p><strong>北岛上部遭遇百年一遇暴雨</strong></p>
<p>据<ahref="/gb/tag/%E6%96%B0%E8%A5%BF%E5%85%B0.md#1">新西兰</a>水与大气国家研究院（NIWA）证实，北岛上部遭遇的这场暴雨是百年一遇的，12小时内降雨量高达225毫米。</p>
<p>自周三凌晨起，奥克兰东南部地区遭遇暴雨洪水袭击，Clevedon镇很多居民被迫撤离家园，大量房产遭受损失，数千头牲畜被洪水卷走。截至周五夜晚，雷电交加的暴风雨已经造成奥克兰和北部地区（Northland）数千户家庭失去电力，多所学校及幼儿中心被关闭，部分地区火车、轮渡和公交服务受到影响。</p>
<p>周五，气象服务部门MetService继续发出警告，提醒上部地区居民保持高度警惕，周日前仍有可能遭遇暴雨袭击。</p>
<p><strong>奥克兰节水建议：</strong></p>
<ul>
<li>将淋浴时间减少2～3分钟，约可节省16升水。</li>
</ul>
<ul>
<li>刷牙时关掉水龙头，约可节省4升水。</li>
</ul>
<ul>
<li>等到洗碗机里待洗碗盘装满时，再开始洗涤。</li>
</ul>
<ul>
<li>等到洗衣机里待洗衣物足量时，再开始洗涤。</li>
</ul>
<p><strong>提请市民注意以下事项：</strong></p>
<ul>
<li>预计暴雨将持续至本周日。</li>
</ul>
<ul>
<li>未来3周内需节约用水。</li>
</ul>
<ul>
<li>大雨期间驾车需格外小心，切勿驾车穿越洪水。</li>
</ul>
<ul>
<li>因多次发生大型泥石流，25号高速Whangamata路段已经关闭。</li>
</ul>
<ul>
<li>如果本周末预定了户外活动，尽可能考虑替代方案。</li>
</ul>
<ul>
<li>如果居住在农村地区，本周末要考虑转移牲畜，特别是洪水多发地带。</li>
</ul>
<ul>
<li>检查排水设施是否畅通，避免在大雨期间引发洪水。</li>
</ul>
<ul>
<li>因遭遇洪水袭击，Tapapakanga地区公园、Hunua地区公园和Waitawa地区公园均被关闭。</li>
</ul>
