  <p>以美国东部时间为准，8月21日中午12:50至下午4:02，美国各地包括最北端的阿拉斯加及位于太平洋海域的夏威夷，都可以看到<ahref="/gb/tag/%E6%97%A5%E9%A3%9F.md#1">日食</a>。有的地区可见全日食，有的为不同程度的日偏食。</p>
  <p>哪些地点可以看到全日食？美国航空航天局说，日全食的轨迹从西向东南，穿过14个州，覆盖110公里（70英里）的范围。</p>
  <p>如最西部的俄勒冈州塞勒姆（Salem）、中部堪萨斯州的堪萨斯城（Kansas City）和伊利诺斯州的卡本代尔市（Carbondale），以及南部田纳西州的纳什维尔市（Nashville）和南卡罗莱纳州查尔斯顿（Charleston）。</p>
  <figure id="attachment_9529564" style="width: 450px" class="wp-caption aligncenter"><ahref="https://i.epochtimes.com/assets/uploads/2017/08/Eclipse-zone.jpg"><img class="wp-image-9529564 size-medium" src="https://i.epochtimes.com/assets/uploads/2017/08/Eclipse-zone-450x253.jpg" alt="美国各地观赏日食信息地图。（航空航天局智能手机应用程序“Eyes Eclipse 2017 Web Application”截图）" width="450" b="253" /></a><figcaption class="wp-caption-text">美国各地观赏日食信息地图。（航空航天局智能手机应用程序“Eyes Eclipse 2017 Web Application”截图）</figcaption></figure>  <p>也就是说，在这些地区，可以观赏到持续时间将近3分钟的日全食。</p>
  <p>届时，如果天气晴朗，在太阳完全变黑的三分钟内，大地将变成一片昏暗，如同夜幕降临，而且气温会降低。而在天地渐渐恢复光明之时，还可以仰望十几分钟之久的日偏食。</p>
  <figure id="attachment_9529534" style="width: 450px" class="wp-caption aligncenter"><ahref="https://i.epochtimes.com/assets/uploads/2017/08/kansas-02.jpg"><img class="size-medium wp-image-9529534" src="https://i.epochtimes.com/assets/uploads/2017/08/kansas-02-450x275.jpg" alt="在中部城市堪萨斯，全日食的景象。（航空航天局智能手机应用程序“Eyes Eclipse 2017 Web Application”截图）" width="450" b="275" /></a><figcaption class="wp-caption-text">在中部城市堪萨斯全日食的景象（航空航天局智能手机应用程序“Eyes Eclipse 2017 Web Application”截图）</figcaption></figure>  <figure id="attachment_9529532" style="width: 450px" class="wp-caption aligncenter"><ahref="https://i.epochtimes.com/assets/uploads/2017/08/kansas-01.jpg"><img class="size-medium wp-image-9529532" src="https://i.epochtimes.com/assets/uploads/2017/08/kansas-01-450x275.jpg" alt="在中部城市堪萨斯，全日食之前的日偏食景象。（航空航天局智能手机应用程序“Eyes Eclipse 2017 Web Application”截图）" width="450" b="275" /></a><figcaption class="wp-caption-text">在中部城市堪萨斯，全日食之前的日偏食景象（航空航天局智能手机应用程序“Eyes Eclipse 2017 Web Application”截图）</figcaption></figure>  <p>航空航天局说，将在日全食的观赏区进行多种<ahref="https://www.nasa.gov/eclipselive" target="_blank">流媒体直播</a>（stream live），在其它仅能看到日偏食地区的人们，也可以通过网络饱览近两个小时的日全食穿过美洲大陆的壮观景象。</p>
  <p>而在其它仅能看到日偏食的地区，人们根据该地距离日全食地带的远近而仰观大小不一的日偏食。如西部的洛杉矶和南部的休斯敦只能看到50%的日偏食，中部的芝加哥可见约90%的日偏食，东北部纽约市可见约80%的日偏食。</p>
  <figure id="attachment_9529542" style="width: 450px" class="wp-caption aligncenter"><ahref="https://i.epochtimes.com/assets/uploads/2017/08/SanDiego-CA.jpg"><img class="size-medium wp-image-9529542 aligncenter" src="https://i.epochtimes.com/assets/uploads/2017/08/SanDiego-CA-450x272.jpg" alt="加州圣地亚哥所见的日偏食。（航空航天局智能手机应用程序“Eyes Eclipse 2017 Web Application”截图）" width="450" b="272" /></a><figcaption class="wp-caption-text">加州圣地亚哥所见的日偏食（航空航天局智能手机应用程序“Eyes Eclipse 2017 Web Application”截图）</figcaption></figure>  <figure id="attachment_9529537" style="width: 450px" class="wp-caption aligncenter"><ahref="https://i.epochtimes.com/assets/uploads/2017/08/NY.jpg"><img class="size-medium wp-image-9529537" src="https://i.epochtimes.com/assets/uploads/2017/08/NY-450x274.jpg" alt="纽约市所见的日偏食景象。（航空航天局智能手机应用程序“Eyes Eclipse 2017 Web Application”截图）" width="450" b="274" /></a><figcaption class="wp-caption-text">纽约市所见的日偏食景象（航空航天局智能手机应用程序“Eyes Eclipse 2017 Web Application”截图）</figcaption></figure>  <p>航空航天局说，可以使用智能手机应用程序“<ahref="https://eyes.jpl.nasa.gov/eyes-on-eclipse-web-app.md#1#" target="_blank">Eyes Eclipse 2017 Web Application</a>”来查找适合自己的观测地点，或浏览各地的日食出现及退去时间。但是，不使用手机的人也可以在这个程序的网页进行搜索浏览，如输入地名“堪萨斯”（kansas），可知道该地位于日全食的观赏地带，下午1:59为日全食的最佳观赏时间，将持续两三分钟。</p>
  <p>此次全美大日食也是全世界关注的天象。全球将有11艘宇宙飞船，包括3艘航空航天局的飞船，捕捉日食的景象；以及超过50颗高空气球对日食景象进行观测和拍照；还有，国际空间站的宇航员将利用太空便利条件摄影记录日食过程。</p>
  <figure id="attachment_6962694" style="width: 450px" class="wp-caption aligncenter"><ahref="https://i.epochtimes.com/assets/uploads/2010/07/1007120239111758.jpg"><img class="wp-image-6962694" src="https://i.epochtimes.com/assets/uploads/2010/07/1007120239111758.jpg" alt="智利复活岛上游客用望远镜观看日食。（AFP）" width="450" b="299" /></a><figcaption class="wp-caption-text">务必注意防护眼镜。图为2010年智利复活岛上游客用各种工具观看350年一遇的南半球日食。(AFP)</figcaption></figure>  <figure id="attachment_6962672" style="width: 450px" class="wp-caption aligncenter"><ahref="https://i.epochtimes.com/assets/uploads/2010/07/1007120239091758.jpg"><img class="wp-image-6962672" src="https://i.epochtimes.com/assets/uploads/2010/07/1007120239091758.jpg" alt="智利复活岛上游客用望远镜观看日食。（AFP）" width="450" b="263" /></a><figcaption class="wp-caption-text">绝对不可以直接看太阳。图为2010年智利复活岛上游客用望远镜观看350年一遇的南半球日食。(AFP)</figcaption></figure>  <p>观赏日食，不论是日全食还是日偏食，最重要的一点是记住用防护眼镜，绝对不要用眼睛直接看太阳，需要使用专用日食墨镜或其它观测工具方可。</p>
  <p>因为阳光的亮度极高，只要有一丝光线进入人眼，也极有可能伤害视觉细胞，甚至造成失明。＃</p>
