<tr><td VALIGN=TOP width="626"><h2 align=center>三星Note 8这九大功能 iPhone不具备</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2017/08/1708232332012357-600x400.jpg" />
<h6>三星电子推出Note系列旗舰新机Galaxy Note 8，搭载无边际屏幕、S Pen及三星史上最高规格的相机。（三星提供）
</h6>
<hr>
<p>【大纪元2017年08月29日讯】（大纪元记者张秉开编译报导）<ahref="/gb/tag/%E4%B8%89%E6%98%9F.md#1">三星</a>最新推出的旗舰手机Galaxy Note 8备受关注，近日已在美国纽约上市。</p>
<p>Galaxy Note 8延续S8系列的双曲面机身设计，支持4G+3G双卡双待机以及Micro SD卡等，但是新增诸如无线充电、转为台式电脑等功能，使其更具实用的特色。</p>
<p>据《商业内幕》8月26日报导，如果与苹果公司的iPhone对比，Galaxy Note 8有九大功能是iPhone不具备的。</p>
<h2>眼睛虹膜扫描加密</h2>
<p>这是一种更快速、更安全及更简单易用的加密功能。Galaxy Note 8也具有指纹或纯密码输入的保护隐私设置。</p>
<h2>无线充电模式</h2>
<p>Galaxy Note 8可以不使用导线而进行无线充电。但据说，iPhone 8将具备这种无线充电模式。</p>
<h2>变为桌面电脑</h2>
<p>Galaxy Note 8具有一种DeX Dock功能，即它可以变成普通电脑的主机，与鼠标、键盘及外接大屏幕显示器连接。</p>
<p>iPhone在此方面逊色不少，只能与电脑显示器或电视匹配，而不能变成主机功能与鼠标和键盘连接。</p>
<h2>手写自动转存动画</h2>
<p>Galaxy Note 8增加一种Live Massage功能，可将用户在触屏上的书写动作自动记录并存储为GIF格式动画影像，之后上传至脸书、推特等。</p>
<h2>双聚焦相机</h2>
<p>Galaxy Note 8的背面相机为双聚焦，可以消除摄像的模糊误差。而只有iPhone 7 plus具有聚焦图像稳定功能。</p>
<h2>模糊摄影模式</h2>
<p>Galaxy Note 8在摄影时可以进行人工聚焦干预，使影像模糊柔化从而达到用户的特定要求。这是iPhone相机不具备的。</p>
<h2>同时显示两个Apps</h2>
<p>这是iPhone不具备的功能，即使大屏幕iPhone 7 plus也没有。</p>
<h2>扩展存储容量</h2>
<p>Galaxy Note 8的内置存储容量为64GB，但是仍可插入高达256GB的外置存储卡。＃</p>
<p>责任编辑：叶紫微</p>
