  <p><strong>旧金山皇宫酒店（Palace Hotel, San Francisco）</strong></p>
  <p>顾名思义，这家拥有142年历史的豪华酒店确实富丽堂皇。玻璃屋顶的室内游泳池，带金色中庭天花板的餐厅和带复古吊灯的大堂适合VIP，或者更具体地说是总统。这家酒店曾接待过八位<ahref="/gb/tag/%E7%BE%8E%E5%9B%BD%E6%80%BB%E7%BB%9F.md#1">美国总统</a>，包括罗斯福和沃伦·哈定（Warren G. Harding）。</p>
  <div class="getty embed image" style="background-color: #fff; display: inline-block; font-family: 'Helvetica Neue',Helvetica,Arial,sans-serif; color: #a7a7a7; font-size: 11px; width: 100%; max-width: 594px;">  <div style="padding: 0; margin: 0; text-align: left;"><a style="color: #a7a7a7; text-decoration: none; font-weight: normal !important; border: none; display: inline-block;" href="http://www.gettyimages.com/detail/474874840" target="_blank">Embed from Getty Images</a></div>  <div style="overflow: hidden; position: relative; height: 0; padding: 71.212121% 0 0 0; width: 100%;"><a style="display: inline-block; position: absolute; top: 0; left: 0; width: 100%; height: 100%; margin: 0;" src="//embed.gettyimages.com/embed?assets=474874840,474874802,474874822,474874882,137559484&amp;et=LKNZWUwVSbRlXj6FdI8uXg&amp;viewMoreLink=off&amp;sig=So-DIpZY-T3SQsh8lCQ3kGmI-5UVZWkWhGdssSDBu4o=&amp;caption=true" width="594" b="423" frameborder="0" scrolling="no"></a></div>  <p style="margin: 0;">  </div>  <p><strong>夏威夷卡哈拉酒店及度假村（The Kahala Hotel &amp; Resort, Hawaii）</strong></p>
  <p>这家豪华酒店建于1964年，位于檀香山一个豪华社区，但仍然营造出一种不拘谨的，家庭友好的氛围。自约翰逊以来的每一任美国总统都曾在这里住过。在酒店的大堂，你会发现曾入住过这个酒店的名人照片。</p>
  <div class="getty embed image" style="background-color: #fff; display: inline-block; font-family: 'Helvetica Neue',Helvetica,Arial,sans-serif; color: #a7a7a7; font-size: 11px; width: 100%; max-width: 594px;">  <div style="padding: 0; margin: 0; text-align: left;"><a style="color: #a7a7a7; text-decoration: none; font-weight: normal !important; border: none; display: inline-block;" href="http://www.gettyimages.com/detail/588705370" target="_blank">Embed from Getty Images</a></div>  <div style="overflow: hidden; position: relative; height: 0; padding: 63.299663% 0 0 0; width: 100%;"><a style="display: inline-block; position: absolute; top: 0; left: 0; width: 100%; height: 100%; margin: 0;" src="//embed.gettyimages.com/embed/588705370?et=uBREBgIUR8FHFSEzV4unVQ&amp;viewMoreLink=off&amp;sig=DZJEM9DnPn3QwxTKvmluWRI4Iku459u_jIr5JuXhElU=&amp;caption=true" width="594" b="376" frameborder="0" scrolling="no"></a></div>  <p style="margin: 0;">  </div>  <div class="getty embed image" style="background-color: #fff; display: inline-block; font-family: 'Helvetica Neue',Helvetica,Arial,sans-serif; color: #a7a7a7; font-size: 11px; width: 100%; max-width: 594px;">  <div style="padding: 0; margin: 0; text-align: left;"><a style="color: #a7a7a7; text-decoration: none; font-weight: normal !important; border: none; display: inline-block;" href="http://www.gettyimages.com/detail/588708460" target="_blank">Embed from Getty Images</a></div>  <div style="overflow: hidden; position: relative; height: 0; padding: 66.498316% 0 0 0; width: 100%;"><a style="display: inline-block; position: absolute; top: 0; left: 0; width: 100%; height: 100%; margin: 0;" src="//embed.gettyimages.com/embed/588708460?et=8tZPeUTnRCZNPAGzoHkhaw&amp;viewMoreLink=off&amp;sig=XkwrCkfFv7xHo804FnEO5llih3gthRg1by5VDl-k7LY=&amp;caption=true" width="594" b="395" frameborder="0" scrolling="no"></a></div>  <p style="margin: 0;">  </div>  <p><strong>旧金山费尔蒙酒店（The Fairmont San Francisco）</strong></p>
  <p>自第27任美国总统威廉．塔夫脱（William Taft）（1909.03.04—1913.03.04）以来的每个美国总统都曾在这家酒店住过，被称为西海岸的白宫。杜鲁门、肯尼迪、克林顿和奥巴马都曾入住过这家酒店。</p>
  <div class="getty embed image" style="background-color: #fff; display: inline-block; font-family: 'Helvetica Neue',Helvetica,Arial,sans-serif; color: #a7a7a7; font-size: 11px; width: 100%; max-width: 594px;">  <div style="padding: 0; margin: 0; text-align: left;"><a style="color: #a7a7a7; text-decoration: none; font-weight: normal !important; border: none; display: inline-block;" href="http://www.gettyimages.com/detail/84604662" target="_blank">Embed from Getty Images</a></div>  <div style="overflow: hidden; position: relative; height: 0; padding: 75.589226% 0 0 0; width: 100%;"><a style="display: inline-block; position: absolute; top: 0; left: 0; width: 100%; height: 100%; margin: 0;" src="//embed.gettyimages.com/embed?assets=84604662,84510903,84536128,84536157,84586198&amp;et=AFy4KQjiQA9TduJyODgfyA&amp;viewMoreLink=off&amp;sig=U8Y7thP_92IOkhQWQ50HA8imxpvJQRhwcsWA8st6_HM=&amp;caption=true" width="594" b="449" frameborder="0" scrolling="no"></a></div>  <p style="margin: 0;">  </div>  <p><strong>科罗拉多州布朗宫酒店和水疗中心（The Brown Palace Hotel and Spa）</strong></p>
  <p>这家最早在1892年就开始营业的酒店，可以说是当地最具标志性的酒店。自罗斯福总统以来（除了奥巴马和柯立芝总统之外）的美国总统都曾入住过这家位于丹佛商业区的酒店。</p>
  <div class="getty embed image" style="background-color: #fff; display: inline-block; font-family: 'Helvetica Neue',Helvetica,Arial,sans-serif; color: #a7a7a7; font-size: 11px; width: 100%; max-width: 594px;">  <div style="padding: 0; margin: 0; text-align: left;"><a style="color: #a7a7a7; text-decoration: none; font-weight: normal !important; border: none; display: inline-block;" href="http://www.gettyimages.com/detail/632218094" target="_blank">Embed from Getty Images</a></div>  <div style="overflow: hidden; position: relative; height: 0; padding: 65.824916% 0 0 0; width: 100%;"><a style="display: inline-block; position: absolute; top: 0; left: 0; width: 100%; height: 100%; margin: 0;" src="//embed.gettyimages.com/embed/632218094?et=NiBEAFkgRaZhSP5EgOObEA&amp;viewMoreLink=off&amp;sig=cFaS1sQSXZ2giUB81T6YrqXfEvSVrUGCBpwUcFCrJ3c=&amp;caption=true" width="594" b="391" frameborder="0" scrolling="no"></a></div>  <p style="margin: 0;">  </div>  <p>更值得一提的是，这家酒店在100多年的历史上从来没有歇业过一天。其悠久的历史可以从酒店的每一个角落得到确认—— 豪华的古老装饰包括古董椅子，红色天鹅绒爱情座椅，古董木制货架上那些放了上百年的东西，以及一个有着闪亮大理石墙壁和华丽木制品的水疗中心。</p>
  <p>酒店有两个总统套房，分别是艾森豪威尔和罗斯福总统套房。</p>
  <p><strong>华盛顿特区威拉德洲际酒店（The Willard InterContinental, Washington, D.C.）</strong></p>
  <p>历史追溯到1816年，这家拥有335间客房的酒店是华盛顿特区和美国最具代表性的豪华酒店之一。也许是因为酒店靠近白宫和国家商场，这是一个颇受美国总统欢迎的酒店（包括亚伯拉罕·林肯都曾光顾过这家酒店）。</p>
  <p>黄金地段是吸引顾客的原因之一，但一个充满了大理石柱子、精美复古的格子天花板和水磨石地板的宏伟大堂，加上法国餐厅、经典酒吧、健身中心和水疗中心，都是这家酒店吸引顾客，尤其是总统级别的顾客的关键。</p>
  <p><strong>亚利桑那巴尔的摩酒店（凤凰城）（Arizona Biltmore, A Waldorf Astoria Hotel, Phoenix）</strong></p>
  <p>这是一家由弗兰克·劳埃德·赖特（Frank Lloyd Wright）的学生在1929年设计的酒店，巧妙地将驼峰山作为其惊人的自然背景。南希和罗纳德·里根总统曾在这里庆祝他们的蜜月（虽然这是在1952年，里跟担任总统前），杰奎琳和肯尼迪总统曾在这家酒店的网球场中打过网球，从胡佛总统到小布什总统，每一位美国总统都曾入住过这家酒店。</p>

  <p>或许你是在2009年听说过亚当斯甘草酒店的，因为当时奥巴马在就职典礼前两周就曾住在这家酒店。但在此之前，亚当斯甘草酒店已经颇有名气。 1884年，约翰·海尔（John Hay，林肯的私人秘书和后任国务卿）和亨利·亚当斯（Henry Adams）在酒店现在所在土地上建立了自己的家园。在1927年，在亚当斯死后，房子被目前的酒店所取代。这家酒店的部分客房可以俯瞰白宫。</p>
  <p><strong>纽约华尔道夫酒店（The Waldorf Astoria, New York City）</strong></p>
  <p>坐落于纽约曼哈顿中城东区的华尔道夫酒店兴建于1931年，是纽约市最具标志性的酒店之一。事实上，自赫伯特·胡佛总统以来的每一位美国总统都曾入住这家酒店（奥巴马总统在华尔道夫被中国公司收购之后终止了这一传统）。</p>
  <p><strong>洛杉矶比佛利希尔顿酒店（The Beverly Hilton, Los Angeles）</strong></p>
  <p>这家拥有560间客房的比佛利山庄酒店由希尔顿（Conrad Hilton）在1955年建成，与其它希尔顿酒店不同。它不仅与好莱坞有不解之缘，每年的好莱坞金球奖都在这里举行，是星光闪耀的所在，同时也是世界政治领袖的首选。自肯尼迪以来的每一任美国总统，直至刚刚卸任的奥巴马都曾入住这家酒店。</p>

  <p><strong>波士顿费尔蒙科普利广场酒店（The Fairmont Copley Plaza, Boston）</strong></p>
  <p>这家位于波士顿后湾街区的383间客房酒店，是前来观光、出差和旅行的人的必选之地。但说到曾来这里住过的美国总统，则可以追溯到塔夫脱总统。无论你是否真的是总统，都可以享受有三个房间、一个壁炉、一个大型大理石浴室和装饰典雅的用餐区的总统套房。</p>
